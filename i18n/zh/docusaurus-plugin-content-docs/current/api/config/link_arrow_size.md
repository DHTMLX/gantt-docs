# link_arrow_size

### Description

@short: 设置链接箭头的大小

@signature: link_arrow_size: number

### Example

~~~jsx
gantt.config.link_arrow_size = 8;
gantt.init("gantt_here");
~~~

**Default value:** 6

### Details

此属性设置链接箭头的HTML元素的大小，但实际上并不会改变箭头的真实尺寸。<br>
要修改箭头的外观，需要通过CSS进行调整。
