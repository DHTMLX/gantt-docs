# link_arrow_size

### Description

@short: 设置链接箭头的大小

@signature: link_arrow_size: number

### Example

~~~jsx
gantt.config.link_arrow_size = 8;
gantt.init("gantt_here");
~~~

**默认值:** 6

### Details

该属性指定 HTML 元素的大小，但不会改变链接箭头的实际大小。
要修改它，需要使用 CSS。
