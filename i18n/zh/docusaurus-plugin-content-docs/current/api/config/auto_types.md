# auto_types
:::info
 此功能仅在PRO版本中提供。 
:::
### Description

@short: 自动将包含子任务的任务切换为项目，并将没有子任务的项目转换回任务

@signature: auto_types: boolean

### Example

~~~jsx
gantt.config.auto_types = true;
~~~

**Default value:** false
