# lightbox_additional_height

### Description

@short: 为 lightbox 添加额外高度

@signature: lightbox_additional_height: number

### Example

~~~jsx
gantt.config.lightbox_additional_height = 90;
~~~

**Default value:** 75

### Details

**lightbox_additional_height** 表示 lightbox 的头部和底部区域的总高度。

### Change log
- 自 v9.0 起弃用
