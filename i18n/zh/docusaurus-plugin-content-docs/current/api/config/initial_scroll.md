# initial_scroll

### Description

@short: 设置时间线区域在初始时是否滚动以显示最早的任务

@signature: initial_scroll: boolean

### Example

~~~jsx
gantt.config.initial_scroll = false;
gantt.init("gantt_here");
~~~

**默认值：** true
