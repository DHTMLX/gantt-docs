# autofit

### Description

@short: 自动调整 grid 的列宽以适应 grid 的宽度

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Default value:** false
