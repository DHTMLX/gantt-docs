# wai_aria_attributes

### Description

@short: 启用 WAI-ARIA 支持，使组件能够被屏幕阅读器正确识别

@signature: wai_aria_attributes: boolean

### Example

~~~jsx
gantt.config.wai_aria_attributes = true;
...
gantt.init("gantt_here");
~~~

**Default value:** true

### Details

在版本 4.1 中引入
