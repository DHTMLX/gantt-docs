# drag_project

:::info
 此功能仅在 PRO 版本中提供。 
:::

### Description

@short: 启用项目类型项的拖放

@signature: drag_project: boolean

### Example

~~~jsx
gantt.config.drag_project = true;
~~~

**默认值：** false

### Related samples
- [可拖拽的项目](https://docs.dhtmlx.com/gantt/samples/08_api/19_draggable_projects.html)

### Details

允许用户将项目及其子任务一起拖放。
