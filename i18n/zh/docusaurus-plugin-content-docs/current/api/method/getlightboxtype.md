# getLightboxType

### Description

@short: 返回当前活动 lightbox 的结构名称

@signature: getLightboxType: () =\> string

### Returns
- `name` - (string) - lightbox 结构的名称

### Example

~~~jsx
gantt.getLightboxType(); -> "task"
~~~
