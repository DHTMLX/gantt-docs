# getLinkCount

### Description

@short: 返回在甘特图中显示的所有依赖链接的数量

@signature: getLinkCount: () =\> number

### Returns
- `count` - (number) - 依赖链接的数量

### Example

~~~jsx
const count = gantt.getLinkCount();
~~~
