# getCalendars

### Description

@short: 获取当前所有已添加到甘特图中的日历

@signature: getCalendars: () =\> any[]

### Returns
- ` calendars` - (array) - 包含所有日历的数组

### Example

~~~jsx
var calendars = gantt.getCalendars();
~~~
