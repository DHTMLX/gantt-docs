# getLinks

### Description

@short: 获取甘特图中显示的所有链接

@signature: getLinks: () =\> Array\<Link\>

### Returns
- `links` - (Array &lt;Link&gt;) - 包含链接对象的数组

### Example

~~~jsx
const links = gantt.getLinks();
~~~

### Details

如需了解如何获取与特定任务相关的所有链接，请参阅 [获取 Link 对象/ID](guides/link-object-operations.md) 和 [getLink](api/method/getlink.md) 文章。
