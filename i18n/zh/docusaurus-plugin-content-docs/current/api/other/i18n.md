# i18n

### Description

@short: 一组用于本地化甘特图的方法集合

@signature: i18n: GanttInternationalization

### Details

**i18n** 对象包含以下方法:

- **addLocale(languageCode, localeObject)** - 允许你向甘特图添加一个新的自定义语言环境
    - **languageCode** - (*string*) 表示语言的两个字母代码
    - **localeObject** - (*object*) 语言环境的详细信息，以对象格式表示
- **setLocale(localeObject: string|object)** - 切换甘特图到指定的语言环境

该方法接受字符串形式的语言代码:
~~~js
gantt.i18n.setLocale("fr");
~~~

或者语言环境对象:

~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"new task"
   }
});
~~~

- **getLocale(languageCode : string)** - 获取指定语言代码对应的语言环境对象
