# config

### Description

@short: 定义日期、刻度、控件的配置选项

@signature: config: GanttConfigOptions

### Example

~~~jsx
//设置时间刻度的格式
gantt.config.date_scale = "%F, %Y";
 
gantt.init("gantt_here");
~~~

### Details

config 对象的属性在根 API 页面的单独章节中描述，请参阅 root API 页面 ["Gantt API: Properties"](api/overview/properties-overview.md)。
