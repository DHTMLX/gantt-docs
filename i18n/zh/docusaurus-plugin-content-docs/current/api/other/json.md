# json

### Description

@short: 处理 JSON 序列化和解析

@signature: json: any

### Example

~~~jsx
var obj = gantt.json; // -> { parse(data){... }}
~~~

### Details

JSON 对象包含一个成员--**parse()** 方法，该方法定义了 dhtmlxGantt 如何处理以 JSON 格式的数据。
