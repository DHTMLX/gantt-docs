# 日期转换模板

- [parse_date](api/template/parse_date.md) - 将日期字符串转换为 Date 对象
- [format_date](api/template/format_date.md) - 将 Date 对象转换为日期字符串。用于将数据发送回服务器

:::note
自从 v9.1.3 版本起，ISO 8601 日期将自动处理，无需覆盖这些模板。参见 [在 ISO 格式下加载日期](guides/loading.md#loading-dates-in-iso-format)。
:::
