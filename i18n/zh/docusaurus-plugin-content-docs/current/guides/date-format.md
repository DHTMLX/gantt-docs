# 日期格式规范

在为日期指定格式时，可以使用下列字符中的任意一个：

- **%y** - 年份以两位数表示 ( _00 到 99_ )；
- **%Y** - 年份以四位数表示 ( _1900-9999_ )；

- **%m** - 月份以带前导零的数字表示 ( _01 到 12_ )；
- **%n** - 月份以不带前导零的数字表示 ( _1 到 12_ )； 
- **%M** - 月份以缩写形式表示 ( _Jan 到 Dec_ )；
- **%F** - 月份以全名表示 ( _January 到 December_ )；

- **%W** - 年度的 ISO-8601 周数。周从周一开始； 
- **%w** - 周数，周的起始日取决于 [start_on_monday](api/config/start_on_monday.md) 配置属性的值，可能从周一或周日开始；


- **%d** - 以带前导零的数字表示的日 ( _01 到 31_ )；
- **%j** - 以不带前导零的数字表示的日 ( _1 到 31_ )；
- **%D** - 日以缩写形式表示 ( _Sun 到 Sat_ )；
- **%l** - 日以全名表示 ( _Sunday 到 Saturday_ )；

- **%h** - 基于 12 小时制的小时 ( _00 到 11_ )；
- **%H** - 基于 24 小时制的小时 ( _00 到 23_ )；
- **%g** - 基于 12 小时制的小时（不带前导零，_1 到 12_）；
- **%G** - 基于 24 小时制的小时（不带前导零，_0 到 23_）；

- **%i** - 以带前导零的数字表示的分钟 ( _00 到 59_ )；
- **%s** - 以带前导零的数字表示的秒 ( _00 到 59_ )；
- **%a** - 显示 **am**（从午夜到中午）和 **pm**（从中午到午夜）；
- **%A** - 显示 **AM**（从午夜到中午）和 **PM**（从中午到午夜）；

例如，如果你想把 2019 年 6 月 1 日表示为 01/06/2019，应指定 "%d/%m/%Y"。
