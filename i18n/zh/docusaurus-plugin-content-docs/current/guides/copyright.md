# 第三方许可证

本文包含 DHTMLX Gantt 库中所包含的许可第三方软件组件的第三方软件通知和/或附加条款与条件。

**Inter font** 在大多数主题中被用作主要字体。该字体通过 CSS `@import` 规则从 dhtmlxgantt.css 动态导入。Inter 字体的本地副本，以及版权声明和许可副本，包含在 `/sources/less/fonts` 的官方发行版中。

**Roboto Google font** 在我们的 Material 皮肤中作为默认字体并未默认包含在库中。但通常按照我们的文档所述手动添加。

此外，在我们的示例中使用了一些开源库，如 **Node.js Express module, body_parser 和 strip_tags**（它们可以在示例的 package.json 中的 backend/package.json 找到）。然而，它们不是 Gantt 库的一部分，仅用于演示目的。
