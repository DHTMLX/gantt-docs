# MSP 项目导出模块更新内容

## 2.1.0.0

* 现已支持在导入和导出的 MSP 及 Primavera 文件中使用自定义日历，包括资源日历。

* 模块现在支持在文件导入时根据单位检索资源分配值。

* 现在可以在导出文件时为资源分配单位指定自定义值。

* 在 Primavera 导出中引入了 **RemainingDuration** 属性，使任务能够在无需额外配置的情况下保持一致的结束日期。

## 2.0.2.0

* MPXJ 库已更新至 11.5.4 版本。

* 现已支持 Primavera 导入和导出时摘要（WBS）任务的自定义属性，即使属性名称相同也能正常工作。要在导出时为摘要任务启用此功能，需要为 **Summary** 属性返回 *true*。

* 修复了 Baseline 0 未能正确导入的问题。

## 2.0.0.1

* 进行了多项内部优化，并添加了导入的 Dockerfile。

## 2.0.0.0

* 模块已从 ASP.NET MVC 迁移到 ASP.NET Core，使其能够在 Linux 系统和 Docker 容器中运行。
