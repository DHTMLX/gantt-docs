# 甘特图界面

甘特图是展示在一段时间内发生的活动的最受欢迎的方式之一。

- 图表中的每个活动都显示为一个条形。
- 图表的 X 轴是时间刻度。
- 图表的 Y 轴是一个网格，显示活动名称及其他与活动相关的信息。网格的第一列是一个树状结构，展示任务的层级关系。你可以通过单击标题附近的 '+'/'-' 图标来收起/展开分支。请注意，甘特图仅显示树中打开的任务。

![gantt_overview_01](/img/gantt_overview_01.png)


## 任务管理

### 新增任务

- 要新增根任务，请点击网格头部的“+”图标。
- 要新增子任务，请在父任务所在行的“+”图标处点击。

### 编辑任务
- 要编辑任务，请在时间线中的条形或网格中的任务记录上双击。然后进行修改，并点击：“保存”以保存更改，“取消”以取消更改。
- 要更改任务的开始日期（结束日期），请在时间线区域拖动该任务。
- 要更改任务的持续时间，请将光标悬停在任务条上，并通过拖动左边界或右边界来调整条的长度。
- 要更改任务的进度，请将光标悬停在任务条上并拖动进度旋钮。

### 删除任务

- 要删除任务，请在时间线中的条形或网格中的任务记录上双击。然后，在打开的编辑表单中点击：“删除”。

## 管理依赖链接

### 新增链接

- 要为任务添加新链接，请将光标悬停在该任务上，将任务开头（结尾）的圆形手柄拖动到另一任务的圆形手柄上。

### 删除链接
- 要删除链接，请在链接上双击，在确认窗口中点击：“删除”。
