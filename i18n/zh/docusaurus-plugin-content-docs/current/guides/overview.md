# 甘特图界面

甘特图是展示一段时间内活动的最流行方式之一。

- 图表中的每个活动都以条形显示。
- 图表的 X 轴为时间轴。
- 图表的 Y 轴为包含活动名称及其他相关信息的网格。网格的第一列是一个树形结构，用于展示任务的层级关系。您可以通过点击标题旁的"+"/"-"图标来展开或收起分支。注意，甘特图只显示树中已展开的任务。

![gantt_overview_01](/img/gantt_overview_01.png)


## 任务管理

### 添加新任务

- 若要添加根任务，请点击网格头部的"+"图标。
- 若要添加子任务，请点击父任务行中的"+"图标。

### 编辑任务
- 若要编辑任务，请在时间线上双击任务条，或在网格中双击任务记录。然后进行修改，并点击:"Save"保存更改，"Cancel"取消更改。
- 若要更改任务的开始（结束）日期，请在时间线区域拖动任务。
- 若要更改任务的持续时间，将鼠标悬停在任务条上，然后拖动条的左侧或右侧边界。
- 若要更改任务进度，将鼠标悬停在任务条上，然后拖动进度滑块。

### 删除任务
- 若要删除任务，请在时间线上双击任务条，或在网格中双击任务记录。然后在打开的编辑表单中点击"Delete"。

## 依赖关系管理


### 添加新依赖

- 若要为任务添加新依赖，将鼠标悬停在任务上，然后从任务开始（结束）处的圆形把手拖动到另一个任务的圆形把手上。

### 删除依赖
- 若要删除依赖，双击该依赖线，在确认窗口中点击"Delete"。
