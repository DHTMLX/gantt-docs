# Vue Gantt

:::note
目前，DHTMLX Gantt 尚未提供原生的 Vue 组件。
不过，您仍然可以通过以下资源，将 DHTMLX Gantt 平滑集成到您的 Vue 应用中。
:::

以下是一些将 DHTMLX Gantt 集成到 Vue 的有用资料:

- [关于如何将 Gantt 集成到 Vue 的分步教程](integrations/vue/howtostart-vue.md)
- [GitHub 上的 Vue Gantt 演示](https://github.com/DHTMLX/vue-gantt-demo)
- [关于在 Vue 中集成 DHTMLX Gantt 的视频教程](https://www.youtube.com/watch?v=IgoXvVSerkE&t=10s)

通过这些资源，您可以轻松地将 DHTMLX Gantt 集成到您的 Vue 项目中，并充分利用其功能。
