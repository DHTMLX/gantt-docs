# AI 工具

本节介绍使用 DHTMLX Gantt 的 AI 辅助开发，以及为基于甘特图的应用添加 AI 功能。

## AI 辅助开发

- [DHTMLX MCP Server](./mcp-server/) - 将 AI 编码工具连接到最新的 DHTMLX 文档和 API 参考。
- [Agent Skills](./agent-skills/) - 在使用 DHTMLX Gantt 时，教授 AI 编码助手（Claude Code、Codex 等）正确的工作模式。
- [Lovable AI](./lovable-ai/) - 将 DHTMLX [React Gantt](./../react/) 集成到 Lovable 生成的应用中，使用提示、知识库和 MCP。

## 应用中的 AI 功能
