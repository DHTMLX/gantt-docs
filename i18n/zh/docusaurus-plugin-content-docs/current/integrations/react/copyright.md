# 第三方许可

此页面列出作为 **DHTMLX React Gantt** 一部分分发的组件所包含的第三方软件通知和许可条款。

DHTMLX React Gantt 是一个捆绑发行版，其中包含 DHTMLX Gantt JavaScript 库。因此，与 DHTMLX Gantt 重新分发的所有第三方组件同样适用于 React Gantt。

本页还列出特定于 React 包装器以及可选的 React 样例应用程序的第三方组件。

## 在 React Gantt 样例应用中使用的组件

DHTMLX React Gantt 可能与一个可选的样例应用一起分发。以下第三方组件仅在样例应用中使用，且并非 React Gantt 库本身的一部分。

这些依赖项在样例应用的 `package.json` 中列出。

### React 生态系统与 UI 库

样例应用使用以下第三方库：

- React
- React DOM
- Ant Design
- Material UI
- Emotion
- React Router
- Day.js
- React Draggable
- Base UI 组件

这些库根据各自作者指定的开源许可证（如 MIT、Apache License 2.0，或其他类似宽松许可证）进行许可。

样例应用仅用于演示目的。对这些组件的许可义务仅在样例应用被重新分发或修改时才适用。
