# KI-Werkzeuge

Dieser Abschnitt behandelt die KI-unterstützte Entwicklung mit DHTMLX Gantt und das Hinzufügen von KI-Funktionen zu Ihrer Gantt-basierten Anwendung.

## KI-gestützte Entwicklung

- [DHTMLX MCP Server](./mcp-server/) - verbinden Sie ein KI-Coding-Tool mit der aktuellen DHTMLX-Dokumentation und API-Referenz.
- [Agent Skills](./agent-skills/) - lehren Sie KI-Coding-Assistenten (Claude Code, Codex und andere) die richtigen Muster für die Arbeit mit DHTMLX Gantt.
- [Lovable AI](./lovable-ai/) - integrieren Sie DHTMLX [React Gantt](integrations/react.md) in eine Lovable-generierte App unter Verwendung von Prompts, Wissensdatenbank und MCP. Der begleitende [Lovable Starter Walkthrough](./lovable-starter-walkthrough/) zeigt, wie man einen vollständigen Supabase-gestützten Projektplaner reproduziert.

## KI-Builder-Beispiele

- [React Gantt Lovable Starter](https://github.com/DHTMLX/react-gantt-lovable-starter) - Mehrprojekt-Planer mit einem Supabase-Backend, vollständig von Anfang bis Ende in Lovable generiert. Enthält die Prompt-Sequenz, die zu deren Aufbau verwendet wurde. [Live-Demo](https://react-gantt-lovable-starter.lovable.app).

## KI-Funktionen in Ihrer App

Wenn Sie KI-Funktionen in Ihre Anwendung integrieren (OpenAI-kompatible APIs, Assistenten, intelligente Vorschläge), sehen Sie:

Guides:

- [Semantische Suche](./semantic-search/) - Fügen Sie Ihrem Gantt-Diagramm eine bedeutungsbasierte Aufgabenerkennung hinzu, indem Sie Einbettungen (Embeddings) und Kosinusähnlichkeit verwenden.
- [AI Assistant](./ai-assistant/) - verbinden Sie einen Chat-Assistenten mit DHTMLX Gantt mittels Tool-Aufrufen und client-seitiger Befehlsausführung.
