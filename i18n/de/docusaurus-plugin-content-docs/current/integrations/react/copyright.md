# Lizenzen von Drittanbietern

Diese Seite listet Hinweise zu Drittanbietersoftware sowie Lizenzbedingungen für Komponenten, die als Teil von **DHTMLX React Gantt** verteilt werden.

DHTMLX React Gantt ist eine gebundene Distribution, die die DHTMLX Gantt JavaScript-Bibliothek enthält. Folglich gelten alle Drittanbieter-Komponenten, die zusammen mit DHTMLX Gantt verteilt werden, auch für React Gantt.

Diese Seite listet zudem Drittanbieter-Komponenten auf, die speziell für das React-Wrapping und für die optionale React-Beispiele-Anwendung gelten.

## Komponenten, die in der React Gantt-Beispiele-Anwendung verwendet werden

DHTMLX React Gantt kann zusammen mit einer optionalen Beispielanwendung verteilt werden. Die folgenden Drittanbieter-Komponenten werden **nur in der Beispielanwendung** verwendet und **sind kein Bestandteil der React Gantt-Bibliothek selbst**.

Diese Abhängigkeiten sind in der package.json der Beispielanwendung aufgeführt.

### React-Ökosystem und UI-Bibliotheken

Die Beispiellanwendung verwendet Drittanbieter-Bibliotheken wie:

- React
- React DOM
- Ant Design
- Material UI
- Emotion
- React Router
- Day.js
- React Draggable
- Base UI components

Diese Bibliotheken sind unter ihren jeweiligen Open-Source-Lizenzen (MIT, Apache License 2.0 oder ähnliche zulässige Lizenzen) lizenziert, wie von ihren Autoren angegeben.

Die Beispielanwendung dient ausschließlich Demonstrationszwecken. Lizenzverpflichtungen für diese Komponenten gelten nur, wenn die Beispielanwendung weiterverteilt oder modifiziert wird.
