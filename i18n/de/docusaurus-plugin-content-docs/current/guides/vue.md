# Vue Gantt

:::note
Im Moment gibt es keine native Vue-Komponente für DHTMLX Gantt. 
Dennoch können Sie DHTMLX Gantt problemlos in Ihre Vue-Anwendungen integrieren, indem Sie die folgenden Ressourcen nutzen.
:::

Hier finden Sie einige hilfreiche Materialien zur Integration von DHTMLX Gantt mit Vue:

- [Schritt-für-Schritt-Anleitung zur Integration von Gantt mit Vue](integrations/vue/howtostart-vue.md)
- [Vue Gantt Demo auf GitHub](https://github.com/DHTMLX/vue-gantt-demo)
- [Videoanleitung zur Integration von DHTMLX Gantt mit Vue](https://www.youtube.com/watch?v=IgoXvVSerkE&t=10s)

Mit diesen Ressourcen ist es einfach, DHTMLX Gantt in Ihre Vue-Projekte einzubinden und die umfangreichen Funktionen optimal zu nutzen.
