# Vue Gantt

:::note
Derzeit gibt es keine native Vue-Komponente für DHTMLX Gantt. 
Sie können DHTMLX Gantt jedoch nach wie vor problemlos in Ihre Vue-Anwendungen integrieren, indem Sie die untenstehenden Materialien verwenden.
:::

Hier sind die empfohlenen Ressourcen zur Integration von DHTMLX Gantt mit Vue:

- [Schritt-für-Schritt-Anleitung zur Integration von Gantt mit Vue](integrations/vue/howtostart-vue.md)
- [Vue Gantt-Demo auf GitHub](https://github.com/DHTMLX/vue-gantt-demo)
- [Video-Tutorial zur Integration von DHTMLX Gantt mit Vue](https://www.youtube.com/watch?v=IgoXvVSerkE&t=10s)

Wenn Sie diese Anleitungen befolgen, können Sie DHTMLX Gantt effektiv in Vue-Projekten verwenden und dessen volle Leistungsfähigkeit nutzen.
