Angular Gantt
==================

:::note
Aktuell gibt es keine native Angular-Komponente für DHTMLX Gantt.
Sie können DHTMLX Gantt jedoch dennoch problemlos in Ihre Angular-Anwendungen integrieren, indem Sie die untenstehenden Materialien nutzen.
:::

Nachfolgend finden Sie einige hilfreiche Ressourcen zur Einbindung von DHTMLX Gantt in Ihre Angular-Projekte:

- [Schritt-für-Schritt-Anleitung zur Integration von Gantt mit Angular](integrations/angular/howtostart-angular.md)
- [Angular Gantt Demo auf GitHub](https://github.com/DHTMLX/angular-gantt-demo)
- [Video-Tutorial zur Integration von DHTMLX Gantt mit Angular](https://www.youtube.com/watch?v=LNVgNVfwzPE&t=3s)

Diese Materialien führen Sie durch den Prozess der Nutzung von DHTMLX Gantt in Angular und ermöglichen es Ihnen, den vollen Funktionsumfang zu nutzen.
