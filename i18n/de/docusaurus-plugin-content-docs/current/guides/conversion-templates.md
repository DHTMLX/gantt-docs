# Vorlagen zur Datumsumwandlung

- [parse_date](api/template/parse_date.md) - konvertiert eine Datumszeichenfolge in ein Date-Objekt
- [format_date](api/template/format_date.md) - konvertiert ein Date-Objekt in eine Datumszeichenfolge. Wird verwendet, um Daten an den Server zurückzusenden

:::note
Seit v9.1.3 werden ISO 8601-Daten automatisch behandelt, ohne diese Vorlagen zu überschreiben. Siehe [Laden von Daten im ISO-Format](guides/loading.md#loading-dates-in-iso-format).
:::
