Vorlagen für die Datumskonvertierung
=======================================

- [parse_date](api/template/parse_date.md) - wandelt einen Datums-String in ein Date-Objekt um
- [format_date](api/template/format_date.md) - wandelt ein Date-Objekt in einen Datums-String um. Dies ist nützlich, wenn Daten zurück an den Server gesendet werden
