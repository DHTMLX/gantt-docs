# Svelte Gantt

:::note
Derzeit gibt es keine native Svelte-Komponente für DHTMLX Gantt. 
Sie können DHTMLX Gantt jedoch weiterhin einfach in Ihre Svelte-Anwendungen integrieren, indem Sie die untenstehenden Materialien verwenden.
:::

Hier sind die empfohlenen Ressourcen zur Integration von DHTMLX Gantt in Svelte:

- [Schritt-für-Schritt-Anleitung zur Integration von Gantt mit Svelte](integrations/svelte/howtostart-svelte.md)
- [Svelte Gantt-Demo auf GitHub](https://github.com/DHTMLX/svelte-gantt-demo)

Wenn Sie diesen Leitfäden folgen, können Sie DHTMLX Gantt effektiv in Svelte-Projekten verwenden und dessen volle Leistungsfähigkeit ausschöpfen.
