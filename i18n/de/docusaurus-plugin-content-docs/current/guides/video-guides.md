# Videoanleitungen

Der Artikel bietet eine kurze Übersicht über Videoanleitungen, die Ihnen helfen sollen, Gantt besser zu verstehen.

Indem Sie diesen Video-Tutorials folgen, können Sie Themen wie das Konfigurieren von Grid-Spalten, das Anpassen von Zellen und Überschriften des Grids sowie das Erstellen eines Standard-Gantt-Diagramms auf verschiedenen serverseitigen Plattformen beherrschen.
