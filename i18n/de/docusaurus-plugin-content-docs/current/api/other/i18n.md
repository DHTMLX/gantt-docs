# i18n

### Description

@short: Eine Sammlung von Methoden zur Lokalisierung des Gantt-Diagramms

@signature: i18n: GanttInternationalization

### Details

Das **i18n**-Objekt beinhaltet die folgenden Methoden:

- **addLocale(languageCode, localeObject)** - ermöglicht das Hinzufügen einer neuen benutzerdefinierten Locale zum Gantt 
    - **languageCode** - (*string*) ein zweistelliger Code, der die Sprache repräsentiert 
    - **localeObject** - (*object*) die Locale-Details im Objektformat
- **setLocale(localeObject: string|object)** - wechselt das Gantt-Diagramm zur angegebenen Locale

Diese Methode akzeptiert entweder einen Sprachcode als string:
~~~js
gantt.i18n.setLocale("fr");
~~~

oder ein Locale-Objekt:

~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"new task"
   }
});
~~~

- **getLocale(languageCode : string)** - ruft das Locale-Objekt für den angegebenen Sprachcode ab
