# i18n

### Description

@short: Eine Sammlung von Methoden zur Lokalisierung des Gantt-Diagramms

@signature: i18n: GanttInternationalization


### Details

Das **i18n** Objekt bietet die folgenden Methoden:

- **addLocale(languageCode, localeObject)** - fügt eine neue benutzerdefinierte Locale in Gantt hinzu
    - **languageCode** - (*string*) ein Sprachcode aus zwei Buchstaben
    - **localeObject** - (*object*) ein Objekt der Locale
- **setLocale(localeObject: string|object)** - aktiviert eine angegebene Locale

Die Methode kann als Parameter entweder einen Sprachcode entgegennehmen:
~~~js
gantt.i18n.setLocale("fr");
~~~

oder ein Objekt der Locale:
~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"new task"
   }
});
~~~


- **getLocale(languageCode : string)** - gibt ein Locale-Objekt zurück, das dem Sprachcode entspricht
