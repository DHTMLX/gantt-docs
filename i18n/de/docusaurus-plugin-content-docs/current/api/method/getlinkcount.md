# getLinkCount

### Description

@short: Gibt die Anzahl aller Abhängigkeits-Links im Gantt-Diagramm zurück

@signature: getLinkCount: () =\> number

### Returns
- ` count` - (number) - die Anzahl der Abhängigkeits-Links

### Example

~~~jsx
const count = gantt.getLinkCount();
~~~
