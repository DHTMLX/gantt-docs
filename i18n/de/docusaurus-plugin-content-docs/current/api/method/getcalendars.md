# getCalendars

### Description

@short: Ruft alle in Gantt hinzugefügten Kalender ab

@signature: getCalendars: () =\> any[]

### Returns
- ` calendars` - (array) - ein Array von Kalendern

### Example

~~~jsx
var calendars = gantt.getCalendars();
~~~
