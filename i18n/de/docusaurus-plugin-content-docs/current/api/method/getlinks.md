# getLinks

### Description

@short: Ruft alle im Gantt-Diagramm angezeigten Links ab

@signature: getLinks: () =\> Array\<Link\>

### Returns
- `links` - (Array &lt;Link&gt;) - ein Array, das Link-Objekte enthält

### Example

~~~jsx
const links = gantt.getLinks();
~~~

### Details

Um zu erfahren, wie man alle Links abruft, die mit einer bestimmten Aufgabe verknüpft sind, siehe die Artikel ["Abrufen des Link-Objekts/der Link-ID"](guides/link-object-operations.md#gettingthelinksrelatedtoacertaintask) und [getLink](api/method/getlink.md).
