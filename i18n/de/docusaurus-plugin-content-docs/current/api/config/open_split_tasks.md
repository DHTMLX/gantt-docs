# open_split_tasks

### Description

@short: Ermöglicht das Erweitern bzw. das Einklappen von Split-Tasks durch Klicken auf den +/- Button

@signature: open_split_tasks: boolean

### Example

~~~jsx
gantt.config.open_split_tasks = true;
~~~
