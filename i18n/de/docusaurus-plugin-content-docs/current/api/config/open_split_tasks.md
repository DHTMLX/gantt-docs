# open_split_tasks

### Description

@short: Ermöglicht das Erweitern oder Zusammenklappen von Split-Tasks durch Klicken auf die +/- Schaltfläche

@signature: open_split_tasks: boolean

### Example

~~~jsx
gantt.config.open_split_tasks = true;
~~~
