# scroll_on_click

### Description

@short: Bestimmt, ob die Timeline beim Klicken auf eine Aufgabe automatisch scrollen soll, um die ausgewählte Aufgabe anzuzeigen

@signature: scroll_on_click: boolean

### Example

~~~jsx
gantt.config.scroll_on_click= false;
gantt.init('gantt_here');
~~~

**Standardwert:** true
