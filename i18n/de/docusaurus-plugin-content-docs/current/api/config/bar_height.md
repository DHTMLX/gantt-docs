# bar_height

### Description

@short: Legt die Höhe der Task-Bars im Timeline-Bereich fest

@signature: bar_height: number | string

### Example

~~~jsx
gantt.config.bar_height = 30;
gantt.init("gantt_here");
~~~

**Standardwert:** "full"
