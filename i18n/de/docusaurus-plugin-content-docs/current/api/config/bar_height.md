# bar_height

### Description

@short: Definiert die Höhe der Task-Bars im Timeline-Bereich

@signature: bar_height: number | string

### Example

~~~jsx
gantt.config.bar_height = 30;
gantt.init("gantt_here");
~~~

**Default value:** "full"
