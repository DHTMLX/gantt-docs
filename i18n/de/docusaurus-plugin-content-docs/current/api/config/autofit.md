# autofit

### Description

@short: Passt die Spalten des Grids automatisch an die Breite des Grids an

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Default value:** false
