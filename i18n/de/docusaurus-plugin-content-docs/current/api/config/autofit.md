# autofit

### Description

@short: Ermöglicht die automatische Anpassung der Grid-Spalten an die Breite des Grids

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Default value:** false
