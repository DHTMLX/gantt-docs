# show_progress

### Description

@short: Aktiviert die Anzeige des Fortschritts in den Aufgabenbalken

@signature: show_progress: boolean

### Example

~~~jsx
gantt.config.show_progress = true;
...
gantt.init("gantt_here");
~~~

**Standardwert:** true
