# show_progress

### Description

@short: Ermöglicht die Anzeige des Fortschritts innerhalb der Task-Bars

@signature: show_progress: boolean

### Example

~~~jsx
gantt.config.show_progress = true;
...
gantt.init("gantt_here");
~~~

**Default value:** true
