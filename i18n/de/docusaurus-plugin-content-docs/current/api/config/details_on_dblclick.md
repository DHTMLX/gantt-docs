# details_on_dblclick

### Description

@short: 'sagt' öffnet die Lightbox nach dem Doppelklick auf eine Aufgabe

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Standardwert:** true
