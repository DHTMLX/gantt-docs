# details_on_dblclick

### Description

@short: Ermöglicht das Öffnen der Lightbox durch Doppelklick auf eine Aufgabe

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Default value:** true
