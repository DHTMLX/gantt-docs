# show_errors

### Description

@short: Aktiviert das Anzeigen von Fehlermeldungen bei unerwartetem Verhalten

@signature: show_errors: boolean

### Example

~~~jsx
gantt.config.show_errors = false;
~~~

**Standardwert:** true
