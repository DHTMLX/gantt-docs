# auto_types
:::info
 Diese Funktion ist nur in der PRO-Edition verfügbar. 
:::
### Description

@short: Schaltet Aufgaben mit Unteraufgaben automatisch in Projekte um und wandelt Projekte ohne Unteraufgaben wieder in Aufgaben zurück.

@signature: auto_types: boolean

### Example

~~~jsx
gantt.config.auto_types = true;
~~~

**Default value:** false
