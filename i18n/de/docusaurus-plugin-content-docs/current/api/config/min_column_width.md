# min_column_width

### Description

@short: Legt die minimale Breite einer Spalte im Timeline-Bereich fest

@signature: min_column_width: number

### Example

~~~jsx
gantt.config.min_column_width = 100;
 
gantt.init("gantt_here");
~~~

**Standardwert:** 70
