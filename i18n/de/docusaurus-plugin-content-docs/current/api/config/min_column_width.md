# min_column_width

### Description

@short: Definiert die minimale Breite einer Spalte im Timeline-Bereich

@signature: min_column_width: number

### Example

~~~jsx
gantt.config.min_column_width = 100;

gantt.init("gantt_here");
~~~

**Default value:** 70
