# readonly

### Description

@short: Aktiviert den Nur-Lese-Modus für das Gantt-Diagramm

@signature: readonly: boolean

### Example

~~~jsx
gantt.config.readonly = true;

gantt.init("gantt_here");
~~~

**Default value:** false
