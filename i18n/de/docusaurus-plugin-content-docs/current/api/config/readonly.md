# readonly

### Description

@short: Aktiviert den Read-only-Modus für das Gantt-Diagramm

@signature: readonly: boolean

### Example

~~~jsx
gantt.config.readonly = true;

gantt.init("gantt_here");
~~~

**Standardwert:** false
