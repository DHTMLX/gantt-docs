# select_task

### Description

@short: Ermöglicht das Auswählen von Tasks im Gantt-Diagramm

@signature: select_task: boolean

### Example

~~~jsx
gantt.config.select_task  = false;
gantt.init("gantt_here");
~~~

**Default value:** true
