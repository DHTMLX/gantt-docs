# select_task

### Description

@short: Ermöglicht die Auswahl von Aufgaben im Gantt-Diagramm

@signature: select_task: boolean

### Example

~~~jsx
gantt.config.select_task  = false;
gantt.init("gantt_here");
~~~

**Standardwert:** true
