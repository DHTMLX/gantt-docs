# open_tree_initially

### Description

@short: Öffnet zu Beginn alle Verzweigungen

@signature: open_tree_initially: boolean

### Example

~~~jsx
gantt.config.open_tree_initially = true;

gantt.init("gantt_here");
~~~

**Standardwert:** false
