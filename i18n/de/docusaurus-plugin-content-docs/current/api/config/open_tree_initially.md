# open_tree_initially

### Description

@short: Öffnet standardmäßig alle Zweige

@signature: open_tree_initially: boolean

### Example

~~~jsx
gantt.config.open_tree_initially = true;

gantt.init("gantt_here");
~~~

**Default value:** false
