# link_line_width

### Description

@short: Passt die Dicke der Abhängigkeitslinks im Timeline-Bereich an

@signature: link_line_width: number

### Example

~~~jsx
gantt.config.link_line_width = 3;
gantt.init("gantt_here");
~~~

**Default value:** 2
