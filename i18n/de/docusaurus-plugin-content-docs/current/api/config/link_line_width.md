# link_line_width

### Description

@short: Legt die Breite der Abhängigkeitsverbindungen im Timeline-Bereich fest

@signature: link_line_width: number

### Example

~~~jsx
gantt.config.link_line_width = 3;
gantt.init("gantt_here");
~~~

**Standardwert:** 2
