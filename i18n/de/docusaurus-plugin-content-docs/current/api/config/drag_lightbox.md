# drag_lightbox

### Description

@short: Ermöglicht das Ziehen der Lightbox über ihre Kopfzeile

@signature: drag_lightbox: boolean

### Example

~~~jsx
gantt.config.drag_lightbox = false;

gantt.init("gantt_here");
~~~

**Default value:** true
