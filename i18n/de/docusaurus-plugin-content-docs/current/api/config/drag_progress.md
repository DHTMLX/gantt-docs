# drag_progress

### Description

@short: Ermöglicht die Anpassung des Task-Fortschritts durch Ziehen des Fortschritts-Knopfs

@signature: drag_progress: boolean

### Example

~~~jsx
gantt.config.drag_progress = false;
gantt.init("gantt_here");
~~~

**Default value:** true
