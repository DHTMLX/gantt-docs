# scale_height

### Description

@short: Passt die Höhe der Zeitskala und des Grid-Headers an

@signature: scale_height: number

### Example

~~~jsx
gantt.config.scale_height = 50;
gantt.init("gantt_here");
~~~

**Default value:** 35
