# scale_height

### Description

@short: Legt die Höhe der Zeitachse und der Kopfzeile des Grids fest

@signature: scale_height: number

### Example

~~~jsx
gantt.config.scale_height = 50;
gantt.init("gantt_here");
~~~

**Standardwert:** 35
