# details_on_create

### Description

@short: 'says' öffnet das Lightbox-Fenster beim Erstellen neuer Ereignisse, indem man auf die '+'-Schaltfläche klickt

@signature: details_on_create: boolean

### Example

~~~jsx
gantt.config.details_on_create = true;
gantt.init("gantt_here");
~~~

**Standardwert:** true
