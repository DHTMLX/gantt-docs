# initial_scroll

### Description

@short: Steuert, ob der Timeline-Bereich zu Beginn so gescrollt wird, dass die früheste Aufgabe angezeigt wird

@signature: initial_scroll: boolean

### Example

~~~jsx
gantt.config.initial_scroll = false;
gantt.init("gantt_here");
~~~

**Default value:** true
