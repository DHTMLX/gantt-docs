# initial_scroll

### Description

@short: Legt fest, ob der Timeline-Bereich anfänglich so gescrollt wird, dass die früheste Aufgabe angezeigt wird

@signature: initial_scroll: boolean

### Example

~~~jsx
gantt.config.initial_scroll = false;
gantt.init("gantt_here");
~~~

**Standardwert:** true
