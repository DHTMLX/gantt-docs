# show_links

### Description

@short: Aktiviert/deaktiviert die Anzeige von Verbindungen im Gantt-Diagramm

@signature: show_links: boolean

### Example

~~~jsx
//versteckt alle Links im Gantt-Diagramm
gantt.config.show_links = false;
 
gantt.init("gantt_here");
~~~

**Standardwert:** true
