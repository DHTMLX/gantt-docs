# link_wrapper_width

### Description

@short: Legt die Breite des Bereichs über dem Link fest, der Klicks empfängt

@signature: link_wrapper_width: number

### Example

~~~jsx
gantt.config.link_wrapper_width = 30;
gantt.init("gantt_here");
~~~

**Standardwert:** 20
