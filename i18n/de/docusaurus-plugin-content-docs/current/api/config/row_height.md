# row_height

### Description

@short: Legt die Standardhöhe für die Zeilen in der Tabelle fest

@signature: row_height: number

### Example

~~~jsx
gantt.config.row_height = 40;

gantt.init("gantt_here");
~~~

**Default value:** 35
