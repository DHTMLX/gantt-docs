# row_height

### Description

@short: Legt die Standardhöhe für Zeilen der Tabelle fest

@signature: row_height: number

### Example

~~~jsx
gantt.config.row_height = 40;
 
gantt.init("gantt_here");
~~~

**Standardwert:** 35
