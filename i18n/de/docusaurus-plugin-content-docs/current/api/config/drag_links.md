# drag_links

### Description

@short: Ermöglicht das Erstellen von Abhängigkeitsverknüpfungen per Drag-and-Drop

@signature: drag_links: boolean

### Example

~~~jsx
gantt.config.drag_links = false;
gantt.init("gantt_here");
~~~

**Standardwert:** true
