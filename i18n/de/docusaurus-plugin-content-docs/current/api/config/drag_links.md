# drag_links

### Description

@short: Ermöglicht das Erstellen von Abhängigkeits-Links durch Drag-and-Drop

@signature: drag_links: boolean

### Example

~~~jsx
gantt.config.drag_links = false;
gantt.init("gantt_here");
~~~

**Default value:** true
