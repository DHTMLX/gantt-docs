# show_empty_state

### Description

@short: Steuert, ob ein Platzhalter im Grid angezeigt wird, wenn es leer ist

@signature: show_empty_state: boolean

### Example

~~~jsx
gantt.config.show_empty_state = true;
~~~

**Default value:** false
