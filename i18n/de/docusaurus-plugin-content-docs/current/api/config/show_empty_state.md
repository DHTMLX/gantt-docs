# show_empty_state

### Description

@short: Definiert, ob das Platzhalter-Element innerhalb des Grid angezeigt wird, wenn es leer ist

@signature: show_empty_state: boolean

### Example

~~~jsx
gantt.config.show_empty_state = true;
~~~

**Standardwert:** false
