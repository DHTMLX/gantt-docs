# 표준 버전 vs PRO 라이브러리 버전

dhtmlxGantt 라이브러리의 표준(Standard) 버전과 PRO 버전의 차이가 궁금하고 PRO 에디션에서 어떤 혜택을 얻을 수 있는지 알고 싶다면 이 가이드가 바로 당신이 찾고 있던 내용입니다.

PRO 에디션은 표준 버전의 모든 기능을 포함할 뿐만 아니라 추가로 유용한 기능들을 제공합니다. 아래 표는 기능 목록을 제공하며 dhtmlxGantt의 두 버전을 비교할 수 있게 해 줍니다.
