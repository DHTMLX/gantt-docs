# MSP 프로젝트 내보내기 모듈의 새로운 기능

## 2.1.0.0

• 가져오기 및 내보내기되는 MSP와 Primavera 파일 모두에서 사용자 지정 달력(리소스 달력 포함)에 대한 지원이 추가되었습니다.

• 파일 가져오기 시 리소스 할당 값을 단위 기준으로 가져오는 기능이 추가되었습니다.

• 파일 내보내기 시 리소스 할당 단위에 대해 사용자 지정 값을 지정할 수 있는 옵션이 추가되었습니다.

• Primavera 내보내기를 위해 **RemainingDuration** 속성이 도입되어, 추가 구성 없이 작업의 종료 날짜를 일관되게 유지할 수 있습니다.

## 2.0.2.0

• MPXJ 라이브러리가 버전 11.5.4로 업데이트되었습니다.

• Primavera 가져오기 및 내보내기 모두에 대해 Summary(WBS) 작업의 사용자 지정 속성 지원이 추가되었습니다. 속성 이름이 동일하더라도 작동합니다. 내보내기 시 Summary 작업에 대해 이를 활성화하려면 **Summary** 속성에 대해 *true*를 반환해야 합니다.

• Baseline 0이 올바르게 가져와지지 않는 문제가 수정되었습니다.

## 2.0.0.1

• 다양한 내부 최적화가 이루어졌으며, 가져온 Dockerfile이 추가되었습니다.

## 2.0.0.0

• 모듈이 ASP.NET MVC에서 ASP.NET Core로 마이그레이션되어 Linux 및 Docker 컨테이너에서 실행할 수 있게 되었습니다.
