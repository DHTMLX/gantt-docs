# 날짜 형식 명세

날짜 형식을 지정할 때 아래 목록의 문자 중 아무 문자나 사용할 수 있습니다:

- **%y** - 연도를 두 자리 숫자로 표시 ( _00에서 99까지_ );
- **%Y** - 연도를 네 자리 숫자로 표시 ( _1900-9999_ );

- **%m** - 앞자리 0이 붙은 월 번호 ( _01에서 12까지_ );
- **%n** - 앞자리 0이 없는 월 번호 ( _1에서 12까지_ ); 
- **%M** - 월을 약어로 표시 ( _Jan에서 Dec까지_ );
- **%F** - 월을 전체 이름으로 표시 ( _January에서 December까지_ );

- **%W** - ISO-8601 기준 연도 주 번호. 주는 월요일에 시작합니다;
- **%w** - 주 번호. 주의 시작은 [start_on_monday](api/config/start_on_monday.md) 설정 값에 따라 월요일 또는 일요일 중 하나입니다

- **%d** - 앞자리 0이 붙은 일을 표시 ( _01에서 31까지_ );
- **%j** - 앞자리 0이 없는 일을 표시 ( _1에서 31까지_ );
- **%D** - 요일의 약어로 표시 ( _일 ~ 토_ );
- **%l** - 요일의 전체 이름으로 표시 ( _일요일에서 토요일까지_ );

- **%h** - 12시간 시계 기준의 시 ( _00에서 11까지_ );
- **%H** - 24시간 시계 기준의 시 ( _00에서 23까지_ );
- **%g** - 12시간 시계의 앞자리 0 없이 시 ( _1에서 12까지_ );
- **%G** - 24시간 시계의 앞자리 0 없이 시 ( _0에서 23까지_ );

- **%i** - 앞자리 0이 붙은 분 ( _00에서 59까지_ );
- **%s** - 앞자리 0이 붙은 초 ( _00에서 59까지_ );
- **%a** - am과 pm을 표시합니다 (자정부터 정오까지는 am, 정오부터 자정까지는 pm);
- **%A** - AM과 PM을 표시합니다 (자정부터 정오까지는 AM, 정오부터 자정까지는 PM).

예를 들어 2019년 6월 1일을 01/06/2019로 표시하려면 "%d/%m/%Y"를 지정하면 됩니다.
