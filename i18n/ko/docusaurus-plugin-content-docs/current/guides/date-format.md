날짜 형식 지정
==============

날짜 형식을 설정할 때 다음 문자들을 사용할 수 있습니다:

- **%y** - 두 자리 연도 ( _00부터 99까지_ );
- **%Y** - 네 자리 연도 ( _1900-9999_ );






- **%m** - 앞에 0이 붙은 월 숫자 ( _01부터 12까지_ );
- **%n** - 앞에 0이 없는 월 숫자 ( _1부터 12까지_ );
- **%M** - 축약된 월 이름 ( _Jan부터 Dec까지_ );
- **%F** - 전체 월 이름 ( _January부터 December까지_ );






- **%W** - ISO-8601 기준의 연중 주 번호, 주의 시작은 월요일;
- **%w** - 주 번호, 주의 시작 요일은 [start_on_monday](api/config/start_on_monday.md) 설정에 따라 월요일 또는 일요일;






- **%d** - 앞에 0이 붙은 일 숫자 ( _01부터 31까지_ );
- **%j** - 앞에 0이 없는 일 숫자 ( _1부터 31까지_ );
- **%D** - 축약된 요일 이름 ( _Sun부터 Sat까지_ );
- **%l** - 전체 요일 이름 ( _Sunday부터 Saturday까지_ );






- **%h** - 앞에 0이 붙은 12시간제 시간 ( _00부터 11까지_ );
- **%H** - 앞에 0이 붙은 24시간제 시간 ( _00부터 23까지_ );
- **%g** - 앞에 0이 없는 12시간제 시간 ( _1부터 12까지_ );
- **%G** - 앞에 0이 없는 24시간제 시간 ( _0부터 23까지_ );






- **%i** - 앞에 0이 붙은 분 ( _00부터 59까지_ );
- **%s** - 앞에 0이 붙은 초 ( _00부터 59까지_ );
- **%a** - **am**(자정부터 정오까지) 또는 **pm**(정오부터 자정까지) 표시;
- **%A** - **AM**(자정부터 정오까지) 또는 **PM**(정오부터 자정까지) 표시.

예를 들어, 2019년 6월 1일을 01/06/2019로 표시하려면 "%d/%m/%Y" 형식을 사용하세요.
