# 날짜 변환 템플릿

- [parse_date](api/template/parse_date.md) - 날짜 문자열을 Date 객체로 변환합니다
- [format_date](api/template/format_date.md) - Date 객체를 날짜 문자열로 변환합니다. 서버로 데이터를 다시 보내는 데 사용됩니다

:::note
버전 9.1.3부터 ISO 8601 날짜는 이러한 템플릿을 재정의하지 않고도 자동으로 처리됩니다. 자세한 내용은 [ISO 형식으로 날짜 로드하기](guides/loading.md#loading-dates-in-iso-format)를 참조하십시오.
:::
