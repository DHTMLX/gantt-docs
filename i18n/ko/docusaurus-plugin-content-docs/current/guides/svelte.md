# Svelte Gantt

:::note
현재 DHTMLX Gantt를 위한 전용 Svelte 컴포넌트는 제공되지 않습니다.
하지만 아래에 나열된 자료를 활용하면 DHTMLX Gantt를 Svelte 앱에 원활하게 추가할 수 있습니다.
:::

DHTMLX Gantt를 Svelte에서 사용하기 위한 유용한 자료는 다음과 같습니다:

- [Gantt와 Svelte 통합에 대한 단계별 튜토리얼](integrations/svelte/howtostart-svelte.md)
- [GitHub의 Svelte Gantt 데모](https://github.com/DHTMLX/svelte-gantt-demo)

이 자료들을 활용하면 DHTMLX Gantt를 Svelte 프로젝트에 통합하여 다양한 기능을 효과적으로 사용할 수 있습니다.
