# 간트 차트의 인터페이스

간트 차트는 일정 기간 동안 발생한 활동을 보여주는 가장 일반적인 방법 중 하나입니다.

- 차트의 각 활동은 바(bar)로 표시됩니다.
- 차트의 X축은 시간 척도입니다.
- 차트의 Y축은 활동의 이름과 기타 활동 관련 정보를 담은 그리드입니다. 그리드의 첫 번째 열은 작업의 계층 구조를 나타내는 트리입니다. 제목 옆의 '+'/'-' 아이콘을 클릭하여 가지를 닫거나 열 수 있습니다. 참고로 간트 차트는 트리에서 열려 있는 작업만 표시합니다.

![gantt_overview_01](/img/gantt_overview_01.png)

## 작업 관리

### 새 작업 추가

- 루트 작업을 추가하려면 그리드 헤더의 "+" 아이콘을 클릭합니다.
- 자식 작업을 추가하려면 상위 작업 행의 "+" 아이콘을 클릭합니다.

### 작업 편집
- 작업을 편집하려면 타임라인의 막대 또는 그리드의 작업 기록에서 더블 클릭합니다. 그런 다음 변경 내용을 입력하고 "저장"을 클릭하여 변경 사항을 저장하고, "취소"를 클릭하여 변경 내용을 취소합니다.
- 작업의 시작일(종료일)을 변경하려면 타임라인 영역에서 작업을 드래그합니다.
- 작업의 기간을 변경하려면 커서를 작업 막대 위에 올리고 왼쪽 또는 오른쪽 경계에서 막대를 드래그합니다.
- 작업의 진행률을 변경하려면 작업 막대 위에 커서를 올리고 진행 핸들을 드래그합니다.

### 작업 삭제
- 작업을 삭제하려면 타임라인의 막대 또는 그리드의 작업 기록을 더블 클릭합니다. 그런 다음 열린 편집 양식에서 "삭제"를 클릭합니다.

## 의존성 링크 관리

### 새 링크 추가

- 작업에 새 링크를 추가하려면 커서를 작업 위에 올리고 작업의 시작(끝)에 있는 둥근 핸들을 다른 작업의 둥근 핸들로 드래그합니다.

### 링크 삭제
- 링크를 삭제하려면 링크를 더블 클릭하고 확인 창에서 "삭제"를 클릭합니다.
