# 간트 차트 인터페이스

간트 차트는 일정 기간 동안 발생한 활동을 보여주는 가장 인기 있는 방법 중 하나입니다.

- 차트의 각 활동은 막대로 표시됩니다.
- 차트의 X축은 시간 축입니다.
- 차트의 Y축은 활동 이름과 기타 활동 관련 정보를 포함하는 그리드입니다. 그리드의 첫 번째 열은 작업의 계층 구조를 나타내는 트리입니다. 제목 옆의 '+'/'-' 아이콘을 클릭하여 브랜치를 닫거나 열 수 있습니다. 참고로, 간트 차트는 트리에서 열린 작업만 표시합니다.

![gantt_overview_01](/img/gantt_overview_01.png)

## 작업 관리

### 새 작업 추가

- 루트 작업을 추가하려면, 그리드 헤더의 "+" 아이콘을 클릭하세요.
- 하위 작업을 추가하려면, 상위 작업 행의 "+" 아이콘을 클릭하세요.

### 작업 편집
- 작업을 편집하려면, 타임라인의 막대나 그리드의 작업 레코드를 더블 클릭하세요. 이후 변경 사항을 입력한 후 "Save"를 클릭하면 변경 사항이 저장되고, "Cancel"을 클릭하면 변경이 취소됩니다.
- 작업의 시작(종료) 날짜를 변경하려면, 타임라인 영역에서 작업을 드래그하세요.
- 작업의 기간을 변경하려면, 작업 막대에 커서를 올리고 왼쪽 또는 오른쪽 경계를 드래그하세요.
- 작업의 진행률을 변경하려면, 작업 막대에 커서를 올리고 진행률 노브를 드래그하세요.

### 작업 삭제
- 작업을 삭제하려면, 타임라인의 막대나 그리드의 작업 레코드를 더블 클릭하세요. 이후 열린 편집 폼에서 "Delete"를 클릭하세요.

## 의존성 링크 관리


### 새 링크 추가

- 작업에 새 링크를 추가하려면, 작업 위에 커서를 올리고 작업 시작(종료) 부분의 원형 핸들을 다른 작업의 원형 핸들로 드래그하세요.

### 링크 삭제
- 링크를 삭제하려면, 링크를 더블 클릭한 후 확인 창에서 "Delete"를 클릭하세요.
