Angular Gantt
==================

:::note
현재 DHTMLX Gantt에 대한 네이티브 Angular 컴포넌트는 제공되지 않습니다.
하지만 아래 자료를 활용하면 Angular 애플리케이션에 DHTMLX Gantt를 쉽게 통합할 수 있습니다.
:::

아래는 Angular 프로젝트에 DHTMLX Gantt를 적용하는 데 도움이 되는 자료들입니다:

- [Angular와 Gantt 통합에 대한 단계별 튜토리얼](integrations/angular/howtostart-angular.md)
- [GitHub의 Angular Gantt 데모](https://github.com/DHTMLX/angular-gantt-demo)
- [DHTMLX Gantt와 Angular 통합에 관한 동영상 튜토리얼](https://www.youtube.com/watch?v=LNVgNVfwzPE&t=3s)

이 자료들은 Angular에서 DHTMLX Gantt를 활용하는 과정을 안내하며, 전체 기능을 최대한 활용할 수 있도록 도와줍니다.
