# Vue Gantt


:::note
현재 DHTMLX Gantt를 위한 네이티브 Vue 컴포넌트는 제공되지 않습니다.  
하지만 아래의 자료들을 활용하면 DHTMLX Gantt를 Vue 애플리케이션에 원활하게 통합할 수 있습니다.
:::

DHTMLX Gantt를 Vue와 통합하는 데 도움이 되는 자료는 다음과 같습니다:

- [Gantt와 Vue 통합에 대한 단계별 튜토리얼](integrations/vue/howtostart-vue.md)
- [GitHub의 Vue Gantt 데모](https://github.com/DHTMLX/vue-gantt-demo)
- [DHTMLX Gantt와 Vue 통합에 관한 비디오 튜토리얼](https://www.youtube.com/watch?v=IgoXvVSerkE&t=10s)

이 자료들을 활용하면 DHTMLX Gantt를 Vue 프로젝트에 손쉽게 도입하여 다양한 기능을 효과적으로 사용할 수 있습니다.
