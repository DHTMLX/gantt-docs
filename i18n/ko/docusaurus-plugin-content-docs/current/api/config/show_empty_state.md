# show_empty_state

### Description

@short: 그리드가 비어 있을 때 플레이스홀더 요소를 그리드 내부에 표시할지 여부를 정의합니다

@signature: show_empty_state: boolean

### Example

~~~jsx
gantt.config.show_empty_state = true;
~~~

**Default value:** false
