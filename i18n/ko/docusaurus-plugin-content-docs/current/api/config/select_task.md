# select_task

### Description

@short: 간트 차트에서 작업을 선택할 수 있도록 합니다

@signature: select_task: boolean

### Example

~~~jsx
gantt.config.select_task  = false;
gantt.init("gantt_here");
~~~

**Default value:** true
