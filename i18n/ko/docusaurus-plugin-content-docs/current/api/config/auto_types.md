# auto_types
:::info
이 기능은 PRO 에디션에서만 사용할 수 있습니다. 
:::
### Description

@short: 하위 작업이 있는 작업을 자동으로 프로젝트로 전환하고, 하위 작업이 없는 프로젝트를 다시 작업으로 변환합니다.

@signature: auto_types: boolean

### Example

~~~jsx
gantt.config.auto_types = true;
~~~

**Default value:** false
