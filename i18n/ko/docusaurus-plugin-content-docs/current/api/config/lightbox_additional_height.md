# lightbox_additional_height

### Description

@short: 라이트박스에 추가 높이를 더합니다

@signature: lightbox_additional_height: number

### Example

~~~jsx
gantt.config.lightbox_additional_height = 90;
~~~

**Default value:** 75

### Details

**lightbox_additional_height**는 라이트박스의 헤더와 푸터 영역의 총 높이를 나타냅니다.
