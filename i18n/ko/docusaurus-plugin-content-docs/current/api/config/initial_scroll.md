# initial_scroll

### Description

@short: 타임라인 영역이 가장 이른 작업을 보여주도록 처음에 스크롤된 상태로 시작할지 제어합니다

@signature: initial_scroll: boolean

### Example

~~~jsx
gantt.config.initial_scroll = false;
gantt.init("gantt_here");
~~~

**Default value:** true
