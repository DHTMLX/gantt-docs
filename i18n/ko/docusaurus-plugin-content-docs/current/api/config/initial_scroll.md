# initial_scroll

### Description

@short: 타임라인 영역이 처음에 스크롤되어 가장 이른 작업을 표시하도록 설정합니다

@signature: initial_scroll: boolean

### Example

~~~jsx
gantt.config.initial_scroll = false;
gantt.init("gantt_here");
~~~

**기본값:** true
