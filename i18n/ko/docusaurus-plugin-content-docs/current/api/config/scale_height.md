# scale_height

### Description

@short: 타임 스케일과 그리드 헤더의 높이를 조절합니다

@signature: scale_height: number

### Example

~~~jsx
gantt.config.scale_height = 50;
gantt.init("gantt_here");
~~~

**Default value:** 35
