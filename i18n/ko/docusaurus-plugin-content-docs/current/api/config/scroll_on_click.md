# scroll_on_click

### Description

@short: 선택된 작업을 표시하기 위해 타임라인 영역이 스크롤될지 여부를 지정합니다

@signature: scroll_on_click: boolean

### Example

~~~jsx
gantt.config.scroll_on_click= false;
gantt.init('gantt_here');
~~~

**기본값:** true
