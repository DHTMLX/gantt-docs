# min_task_grid_row_height

### Description

@short: 작업 행 크기 조절 시 최소 높이를 정의합니다

@signature: min_task_grid_row_height: number

### Example

~~~jsx
gantt.config.min_task_grid_row_height = 45;
~~~

**Default value:** 30

### Details

이 설정은 [gantt.config.resize_rows](api/config/resize_rows.md)가 활성화(*true*)된 경우에만 적용됩니다.
