# bar_height

### Description

@short: 타임라인 영역의 작업 바 높이를 설정합니다

@signature: bar_height: number | string

### Example

~~~jsx
gantt.config.bar_height = 30;
gantt.init("gantt_here");
~~~

**Default value:** "full"
