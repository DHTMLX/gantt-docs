# preserve_scroll

### Description

@short: 간트 차트를 다시 그리는 동안 세로 및 가로 스크롤의 현재 위치를 유지합니다

@signature: preserve_scroll: boolean

### Example

~~~jsx
gantt.config.preserve_scroll = false;

gantt.init("gantt_here");
~~~

**기본값:** true
