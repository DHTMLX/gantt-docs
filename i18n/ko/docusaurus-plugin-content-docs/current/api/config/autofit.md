# autofit

### Description

@short: 그리드의 너비에 맞게 그리드의 열 크기를 자동으로 조정합니다

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Default value:** false
