# drag_lightbox

### Description

@short: 라이트박스의 헤더를 드래그하여 이동할 수 있도록 허용합니다.

@signature: drag_lightbox: boolean

### Example

~~~jsx
gantt.config.drag_lightbox = false;

gantt.init("gantt_here");
~~~

**Default value:** true
