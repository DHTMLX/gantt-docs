# drag_links

### Description

@short: 드래그 앤 드롭을 통해 의존성 링크를 생성할 수 있습니다.

@signature: drag_links: boolean

### Example

~~~jsx
gantt.config.drag_links = false;
gantt.init("gantt_here");
~~~

**Default value:** true
