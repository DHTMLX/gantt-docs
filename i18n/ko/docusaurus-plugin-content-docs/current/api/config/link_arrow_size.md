# link_arrow_size

### Description

@short: 링크 화살표의 크기를 설정합니다

@signature: link_arrow_size: number

### Example

~~~jsx
gantt.config.link_arrow_size = 8;
gantt.init("gantt_here");
~~~

**Default value:** 6

### Details

이 속성은 링크 화살표의 HTML 요소 크기를 설정하지만, 실제 화살표의 크기를 변경하지는 않습니다.<br>
화살표의 외관을 변경하려면 CSS 조정이 필요합니다.
