# show_links

### Description

@short: 간트 차트에서 링크 표시를 켜거나 끕니다.

@signature: show_links: boolean

### Example

~~~jsx
// 간트 차트에서 모든 링크를 숨깁니다.
gantt.config.show_links = false;

gantt.init("gantt_here");
~~~

**Default value:** true
