# row_height

### Description

@short: 테이블 내 행들의 기본 높이를 설정합니다.

@signature: row_height: number

### Example

~~~jsx
gantt.config.row_height = 40;
 
gantt.init("gantt_here");
~~~

**Default value:** 35
