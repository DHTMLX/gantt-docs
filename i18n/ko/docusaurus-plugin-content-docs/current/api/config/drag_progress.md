# drag_progress

### Description

@short: 진행 표시 노브를 드래그하여 작업 진행률을 조정할 수 있습니다.

@signature: drag_progress: boolean

### Example

~~~jsx
gantt.config.drag_progress = false;
gantt.init("gantt_here");
~~~

**Default value:** true
