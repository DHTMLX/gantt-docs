# open_split_tasks

### Description

@short: +/- 버튼을 클릭하여 분할 작업(split tasks)을 확장하거나 축소할 수 있습니다.

@signature: open_split_tasks: boolean

### Example

~~~jsx
gantt.config.open_split_tasks = true;
~~~
