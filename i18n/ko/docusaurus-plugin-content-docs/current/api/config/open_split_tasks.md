# open_split_tasks

### Description

@short: 더하기/빼기 버튼을 클릭하여 분할 작업을 확장/축소할 수 있는 기능을 활성화합니다

@signature: open_split_tasks: boolean

### Example

~~~jsx
gantt.config.open_split_tasks = true;
~~~
