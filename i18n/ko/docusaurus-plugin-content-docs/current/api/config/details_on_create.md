# details_on_create

### Description

@short: '+' 버튼을 클릭하여 새 이벤트를 생성할 때 라이트박스가 자동으로 열리도록 설정합니다.

@signature: details_on_create: boolean

### Example

~~~jsx
gantt.config.details_on_create = true;
gantt.init("gantt_here");
~~~

**Default value:** true
