# drag_move

### Description

@short: 드래그 앤 드롭을 사용하여 작업을 재배치할 수 있도록 허용합니다.

@signature: drag_move: boolean

### Example

~~~jsx
gantt.config.drag_move = false;
gantt.init("gantt_here");
~~~

**Default value:** true
