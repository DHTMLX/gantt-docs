# link_wrapper_width

### Description

@short: 링크 주변의 클릭 가능한 영역의 너비를 정의합니다.

@signature: link_wrapper_width: number

### Example

~~~jsx
gantt.config.link_wrapper_width = 30;
gantt.init("gantt_here");
~~~

**Default value:** 20
