# link_line_width

### Description

@short: 타임라인 섹션에서 의존성 링크의 두께를 조절합니다

@signature: link_line_width: number

### Example

~~~jsx
gantt.config.link_line_width = 3;
gantt.init("gantt_here");
~~~

**Default value:** 2
