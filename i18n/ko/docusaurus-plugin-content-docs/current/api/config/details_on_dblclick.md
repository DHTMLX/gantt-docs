# details_on_dblclick

### Description

@short: 작업을 더블 클릭하여 라이트박스를 열 수 있도록 설정합니다.

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Default value:** true
