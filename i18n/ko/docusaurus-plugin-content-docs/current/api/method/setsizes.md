# setSizes

### Description

@short: Gantt 차트의 크기를 조정합니다

@signature: setSizes: () =\> void

### Example

~~~jsx
gantt.setSizes();
~~~

### Details

이 메서드는 매개변수를 받지 않으며, 차트의 HTML 컨테이너와 동일한 크기로 항상 Gantt 차트를 크기 조정합니다.
