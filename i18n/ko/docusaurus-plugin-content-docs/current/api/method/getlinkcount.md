# getLinkCount

### Description

@short: Gantt 차트에 표시된 모든 의존성 링크의 수를 반환합니다

@signature: getLinkCount: () =\> number

### Returns
- ` count` - (number) - 의존성 링크의 수

### Example

~~~jsx
const count = gantt.getLinkCount();
~~~
