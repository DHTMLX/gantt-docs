# getLinkCount

### Description

@short: 현재 간트 차트에 표시된 의존성 링크의 개수를 반환합니다.

@signature: getLinkCount: () =\> number

### Returns
- ` count` - (number) - 의존성 링크의 총 개수

### Example

~~~jsx
const count = gantt.getLinkCount();
~~~
