# focus

### Description

@short: 간트에 focus를 활성화합니다.

@signature: focus: () =\> void

### Example

~~~jsx
gantt.focus();
~~~

### Related samples
- [Keyboard Navigation](https://docs.dhtmlx.com/gantt/samples/02_extensions/16_keyboard_navigation.html)
- [Keyboard Navigation - navigate cells](https://docs.dhtmlx.com/gantt/samples/02_extensions/17_keyboard_navigation_cell.html)

### Details

:::note
note 이 메서드는 **keyboard_navigation** 확장에 속하므로, 먼저 [keyboard_navigation](guides/extensions-list.md#keyboardnavigation) 플러그인을 활성화해야 합니다. 자세한 내용은 [키보드 내비게이션](guides/keyboard-navigation.md) 문서를 참고하세요. 
:::
