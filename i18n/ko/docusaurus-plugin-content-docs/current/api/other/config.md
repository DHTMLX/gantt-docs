# config

### Description

@short: 날짜, 스케일, 컨트롤에 대한 설정 옵션을 정의합니다.

@signature: config: GanttConfigOptions

### Example

~~~jsx
//시간 스케일의 포맷을 설정합니다.
gantt.config.date_scale = "%F, %Y";
 
gantt.init("gantt_here");
~~~

### Details

config 객체의 속성들은 메인 API 페이지의 별도 섹션 ["Gantt API: Properties"](api/overview/properties-overview.md)에서 자세히 설명되어 있습니다.
