# i18n

### Details

**i18n** 객체에는 다음과 같은 메서드가 포함되어 있습니다:

- **addLocale(languageCode, localeObject)** - 간트에 새로운 커스텀 로케일을 추가할 수 있습니다.
    - **languageCode** - (*string*) 언어를 나타내는 두 글자 코드
    - **localeObject** - (*object*) 로케일 세부 정보를 담은 객체
- **setLocale(localeObject: string|object)** - 간트 차트를 지정한 로케일로 전환합니다.

이 메서드는 문자열로 된 언어 코드 또는 로케일 객체를 인자로 받을 수 있습니다:
~~~js
gantt.i18n.setLocale("fr");
~~~

또는 로케일 객체를 사용할 수도 있습니다:

~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"new task"
   }
});
~~~

- **getLocale(languageCode : string)** - 지정한 언어 코드에 해당하는 로케일 객체를 반환합니다.
