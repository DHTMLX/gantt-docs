# AI 도구

이 섹션은 DHTMLX Gantt를 사용한 AI 보조 개발 및 Gantt 기반 애플리케이션에 AI 기능을 추가하는 방법을 다룹니다.

## AI 보조 개발

- [DHTMLX MCP Server](./mcp-server/) - 최신 DHTMLX 문서 및 API 참조에 AI 코딩 도구를 연결합니다.
- [Agent Skills](./agent-skills/) - AI 코딩 어시스턴트(Claude Code, Codex 등)에게 DHTMLX Gantt와 함께 작동하는 올바른 패턴을 가르칩니다.
- [Lovable AI](./lovable-ai/) - Lovable로 생성된 앱에 프롬프트, 지식 베이스 및 MCP를 사용하여 DHTMLX [React Gantt](./../react/)를 통합합니다.

## 앱 내의 AI 기능

애플리케이션에 AI 기능(OpenAI 호환 API, 어시스턴트, 스마트 제안 등)을 추가하는 경우 아래를 참조하십시오:

가이드:

- [Semantic Search](./semantic-search/) - 임베딩과 코사인 유사도를 사용하여 의미 기반 작업 검색을 Gantt 차트에 추가합니다.

- [Gantt Theme Builder AI Demo](https://github.com/DHTMLX/gantt-theme-builder-ai-demo)
