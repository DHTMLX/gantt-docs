# AI 도구

이 섹션은 DHTMLX Gantt를 사용한 AI 지원 개발 및 Gantt 기반 앱에 AI 기능을 추가하는 방법을 다룹니다.

## AI 지원 개발

- [DHTMLX MCP Server](./mcp-server/) - 최신 DHTMLX 문서 및 API 레퍼런스에 AI 코딩 도구를 연결합니다.
- [Agent Skills](./agent-skills/) - Claude Code, Codex 등 AI 코딩 어시스턴트에게 DHTMLX Gantt와 함께 작동하는 올바른 패턴을 가르칩니다.
- [Lovable AI](./lovable-ai/) - 프롬프트, 지식 베이스, MCP를 사용하여 Lovable로 생성된 앱에 DHTMLX [React Gantt](integrations/react.md)를 통합합니다. 동반된 [Lovable Starter Walkthrough](./lovable-starter-walkthrough/)은 Supabase를 백엔드로 하는 전체 프로젝트 플래너를 재현하는 방법을 보여줍니다.

## AI 빌더 쇼케이스

- [React Gantt Lovable Starter](https://github.com/DHTMLX/react-gantt-lovable-starter) - Supabase 백엔드를 가진 다중 프로젝트 플래너로 Lovable에서 엔드투엔드로 생성되었습니다. 이를 생성하는 데 사용된 프롬프트 시퀀스를 포함합니다. [라이브 데모](https://react-gantt-lovable-starter.lovable.app).

## 앱 내 AI 기능

애플리케이션에 AI 기능(OpenAI와 호환되는 API, 어시스턴트, 스마트 제안 등)을 추가하는 경우 아래를 참조하십시오:

가이드:

- [Semantic Search](./semantic-search/) - 임베딩과 코사인 유사도를 사용하여 간트 차트에 의미 기반 작업 검색을 추가합니다.
- [AI Assistant](./ai-assistant/) - 도구 호출과 클라이언트 측 명령 실행을 사용하여 DHTMLX Gantt에 채팅 어시스턴트를 연결합니다.

- [Gantt Maker AI Demo](https://github.com/DHTMLX/gantt-maker-ai-demo)
