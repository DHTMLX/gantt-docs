# 제3자 라이선스

이 페이지에는 **DHTMLX React Gantt** 구성 요소의 일부분으로 배포되는 타사 소프트웨어 공지 및 라이선스 조항이 나와 있습니다.

DHTMLX React Gantt는 DHTMLX Gantt 자바스크립트 라이브러리를 포함하는 번들 배포판입니다. 그 결과, DHTMLX Gantt와 함께 재배포되는 모든 타사 구성 요소도 React Gantt에 적용됩니다.

이 페이지는 또한 React 래퍼 및 선택적 React 샘플 애플리케이션에 특정한 타사 구성 요소를 추가로 나열합니다.

## React Gantt 샘플 애플리케이션에서 사용되는 구성 요소

DHTMLX React Gantt는 선택적 샘플 애플리케이션과 함께 배포될 수 있습니다. 아래의 타사 구성 요소들은 **샘플 애플리케이션에서만 사용되며** **React Gantt 라이브러리 자체의 일부가 아닙니다.**

이 의존성은 샘플 애플리케이션의 `package.json`에 나와 있습니다.
