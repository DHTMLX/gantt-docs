# Лицензии сторонних разработчиков

Эта страница перечисляет уведомления об использовании стороннего программного обеспечения и условия лицензирования компонентов, распространяемых в составе **DHTMLX React Gantt**.

DHTMLX React Gantt является объединённой дистрибуцией, которая включает библиотеку DHTMLX Gantt JavaScript. В результате все сторонние компоненты, распространяемые вместе с DHTMLX Gantt, также применяются к React Gantt.

Эта страница дополнительно перечисляет сторонние компоненты, специфичные для обёртки React и для необязательного примера приложения на React.

## Компоненты, используемые в приложении с примерами React Gantt

DHTMLX React Gantt может быть распространяем вместе с необязательным приложением примеров. Следующие сторонние компоненты используются **только в приложении примеров** и не являются частью самой библиотеки React Gantt.

Эти зависимости перечислены в файле `package.json` приложения примеров.

### Экосистема React и UI-библиотеки

- Ant Design
- Material UI
- Emotion
