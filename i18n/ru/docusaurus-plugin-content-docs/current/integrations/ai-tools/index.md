# Инструменты ИИ

Этот раздел охватывает разработку с использованием ИИ в DHTMLX Gantt и внедрение функций ИИ в приложение на основе Gantt.

## Разработка с поддержкой ИИ

- [DHTMLX MCP Server](./mcp-server/) - соединить инструмент кодирования на основе ИИ с актуальной документацией DHTMLX и справкой по API.
- [Agent Skills](./agent-skills/) - обучить ИИ-ассистентов по программированию (Claude Code, Codex и др.) правильным шаблонам взаимодействия с DHTMLX Gantt.
- [Lovable AI](./lovable-ai/) - интегрировать DHTMLX [React Gantt](integrations/react.md) в приложение, созданное Lovable, с использованием подсказок, базы знаний и MCP. Сопутствующий [Lovable Starter Walkthrough](./lovable-starter-walkthrough/) демонстрирует, как воспроизвести полноценный планировщик проекта на базе Supabase.

## Демонстрации AI-builder

- [React Gantt Lovable Starter](https://github.com/DHTMLX/react-gantt-lovable-starter) - много-проектный планировщик с бэкендом на Supabase, полностью созданный в Lovable. Включает последовательность подсказок, использованную при его создании. [Демонстрация в реальном времени](https://react-gantt-lovable-starter.lovable.app).

## Возможности ИИ внутри вашего приложения

Если вы добавляете возможности ИИ в ваше приложение (API, совместимые с OpenAI, помощники, умные подсказки), ознакомьтесь с:

### Руководства

- [Семантический поиск](./semantic-search/) - добавить поиск задач на основе смысла в вашем графике Gantt, используя эмбеддинги и косинусное сходство.
- [AI Assistant](./ai-assistant/) - подключить чат-ассистента к DHTMLX Gantt с использованием вызовов инструментов и выполнения команд на стороне клиента.
