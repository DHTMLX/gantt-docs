# Инструменты ИИ

Этот раздел охватывает разработку с использованием ИИ в DHTMLX Gantt и добавление функций ИИ в приложение на основе Gantt.

## Разработка с использованием ИИ

- [DHTMLX MCP Server](./mcp-server/) - подключить инструмент кодирования на базе ИИ к актуальной документации DHTMLX и справке по API.
- [Agent Skills](./agent-skills/) - научить AI-ассистентов по кодированию (Claude Code, Codex и другие) правильным паттернам для работы с DHTMLX Gantt.
- [Lovable AI](./lovable-ai/) - интегрировать DHTMLX [React Gantt](./../react/) в Lovable-сгенерированное приложение с использованием подсказок, Базы знаний и MCP.
