# Навыки агента

ИИ-помощники по кодированию, такие как Claude Code или Codex, могут генерировать код DHTMLX Gantt, но часто совершают ошибки в специализированных API: неверные имена свойств, отсутствие импортов CSS, некорректные сигнатуры колбэков, или смешивание несовместимых паттернов данных. Навыки агента решают это, обучая помощника правильным паттернам и известным подводным камням до того, как он напишет код.

В отличие от [MCP server](integrations/ai-tools/mcp-server.md), который предоставляет справочник API в режиме реального времени, навыки фокусируются на паттернах интеграции, точках принятия решений и предотвращении сбоев.

**GitHub:** [DHTMLX/skills](https://github.com/DHTMLX/skills)

## Доступные навыки

### DHTMLX React Gantt

Оснащает интеграцию `@dhtmlx/trial-react-gantt` и `@dhx/react-gantt` в React-приложения. Навык помогает помощнику добавить Gantt в проект и корректно его настроить, подключить CRUD-операции и управлять темами, чтобы Gantt повторно использовал тему приложения, а не уходил в рассинхрон. Также включает известные подводные камни, взятые из реальных проектов, и указывает помощнику проверять незнакомые API через [DHTMLX MCP server](integrations/ai-tools/mcp-server.md), а не гадать.

Файлы навыков представлены в виде читаемого Markdown — вы можете точно ознакомиться с теми правилами, которым будет следовать ваш помощник, в [репозитории GitHub](https://github.com/DHTMLX/skills/tree/main/dhtmlx-react-gantt).

## Установка

```bash
npx skills add DHTMLX/skills --skill dhtmlx-react-gantt
```

### Ручная установка

Клонируйте или скачайте репозиторий [DHTMLX/skills](https://github.com/DHTMLX/skills) и скопируйте папку `dhtmlx-react-gantt` в каталог навыков вашего проекта (например, `.claude/skills/` для Claude Code, `.cursor/skills/` для Cursor).

## Использование навыков с MCP

Навыки и MCP сервер дополняют друг друга. MCP предоставляет вашему помощнику доступ к документации API в режиме реального времени — сигнатуры методов, значения свойств, параметры конфигурации. Навыки обучают его интеграционным паттернам, которые предотвращают типичные ошибки. Чтобы получить наилучшие результаты, используйте оба подхода. См. [DHTMLX MCP server](integrations/ai-tools/mcp-server.md) для инструкций по настройке.
