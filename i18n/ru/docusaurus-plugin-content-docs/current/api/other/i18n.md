# i18n

### Description

@short: Набор методов локализации диаграммы Ганта

@signature: i18n: GanttInternationalization


### Details

Объект **i18n** предоставляет следующие методы:

- **addLocale(languageCode, localeObject)** - добавляет новую настраиваемую локаль в Gantt 
    - **languageCode** - (*string*) двухсимвольный код языка 
    - **localeObject** - (*object*) объект локали
- **setLocale(localeObject: string|object)** - активирует указанную локаль

Метод может принимать в качестве параметра либо языковой код:
~~~js
gantt.i18n.setLocale("fr");
~~~

или объект локали:

~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"новая задача"
   }
});
~~~

- **getLocale(languageCode : string)** - возвращает объект локали по языковому коду
