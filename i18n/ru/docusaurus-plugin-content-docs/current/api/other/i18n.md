# i18n

### Description

@short: Коллекция методов, предназначенных для локализации Gantt chart

@signature: i18n: GanttInternationalization

### Details

Объект **i18n** включает следующие методы:

- **addLocale(languageCode, localeObject)** - позволяет добавить новую пользовательскую локаль в Gantt 
    - **languageCode** - (*string*) двухбуквенный код, обозначающий язык 
    - **localeObject** - (*object*) объект с деталями локали
- **setLocale(localeObject: string|object)** - переключает Gantt chart на указанную локаль

Этот метод принимает либо строку с языковым кодом:
~~~js
gantt.i18n.setLocale("fr");
~~~

либо объект локали:

~~~js
gantt.i18n.setLocale({
   labels: {
       new_task:"new task"
   }
});
~~~

- **getLocale(languageCode : string)** - возвращает объект локали для указанного языкового кода
