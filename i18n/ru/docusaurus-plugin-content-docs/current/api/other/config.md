# config

### Description

@short: Определяет параметры конфигурации для дат, шкалы, элементов управления

@signature: config: GanttConfigOptions

### Example

~~~jsx
//устанавливает формат временной шкалы
gantt.config.date_scale = "%F, %Y";
 
gantt.init("gantt_here");
~~~

### Details

Свойства объекта config подробно описаны в отдельном разделе на главной странице API ["Gantt API: Properties"](api/overview/properties-overview.md).
