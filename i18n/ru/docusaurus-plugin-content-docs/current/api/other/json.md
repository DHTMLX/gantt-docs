# json

### Description

@short: Обрабатывает сериализацию и парсинг JSON

@signature: json: any

### Example

~~~jsx
var obj = gantt.json; // -> { parse(data){... 
:::
~~~

### Details

Объект JSON включает в себя один член - метод **parse()**, который определяет, как dhtmlxGantt обрабатывает данные, форматированные в виде JSON.
