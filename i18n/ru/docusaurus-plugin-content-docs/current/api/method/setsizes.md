# setSizes

### Description

@short: Изменяет размер диаграммы Ганта

@signature: setSizes: () =\> void

### Example

~~~jsx
gantt.setSizes();
~~~

### Details

Метод не принимает никаких параметров — он всегда устанавливает размер диаграммы Ганта равным размеру HTML-контейнера диаграммы.
