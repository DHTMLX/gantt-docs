# setSizes

### Description

@short: Изменяет размер диаграммы Ганта

@signature: setSizes: () =\> void

### Example

~~~jsx
gantt.setSizes();
~~~

### Details

Этот метод не принимает параметров. Он просто изменяет размер диаграммы Ганта, чтобы она соответствовала размерам её HTML контейнера.
