# getCalendars

### Description

@short: Получает все календари, которые в данный момент добавлены в Gantt

@signature: getCalendars: () =\> any[]

### Returns
- ` calendars` - (array) - массив, содержащий все календари

### Example

~~~jsx
var calendars = gantt.getCalendars();
~~~

### Details

введено в версии 4.2
