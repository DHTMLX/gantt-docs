# getLinkCount

### Description

@short: Возвращает общее число связей зависимостей, представленных на диаграмме Ганта

@signature: getLinkCount: () => number

### Returns
- `count` - (number) - количество связей зависимостей

### Example

~~~jsx
const count = gantt.getLinkCount();
~~~
