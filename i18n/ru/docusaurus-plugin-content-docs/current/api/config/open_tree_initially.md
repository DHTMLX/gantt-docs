# open_tree_initially

### Description

@short: Открывает все ветви изначально

@signature: open_tree_initially: boolean

### Example

~~~jsx
gantt.config.open_tree_initially = true;

gantt.init("gantt_here");
~~~

**Значение по умолчанию:** false
