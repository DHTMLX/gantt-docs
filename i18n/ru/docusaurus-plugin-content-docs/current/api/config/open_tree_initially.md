# open_tree_initially

### Description

@short: Открывает все ветви по умолчанию

@signature: open_tree_initially: boolean

### Example

~~~jsx
gantt.config.open_tree_initially = true;

gantt.init("gantt_here");
~~~

**Default value:** false
