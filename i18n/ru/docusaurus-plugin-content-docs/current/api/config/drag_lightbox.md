# drag_lightbox

### Description

@short: Позволяет перетащить lightbox за заголовком

@signature: drag_lightbox: boolean

### Example

~~~jsx
gantt.config.drag_lightbox = false;

gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
