# drag_progress

### Description

@short: Позволяет изменять прогресс задачи путем перетаскивания регулятора прогресса

@signature: drag_progress: boolean

### Example

~~~jsx
gantt.config.drag_progress = false;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
