# drag_progress

### Description

@short: Позволяет регулировать прогресс задачи с помощью перетаскивания ручки прогресса

@signature: drag_progress: boolean

### Example

~~~jsx
gantt.config.drag_progress = false;
gantt.init("gantt_here");
~~~

**Default value:** true
