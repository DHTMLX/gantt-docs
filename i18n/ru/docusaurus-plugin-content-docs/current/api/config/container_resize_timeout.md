# container_resize_timeout

### Description

@short: Задаёт время задержки (в миллисекундах) перед перерисовкой gantt после изменения размера контейнера

@signature: container_resize_timeout: number

### Example

~~~jsx
gantt.config.container_resize_timeout = 300;
~~~

**Default value:** 20

### Change log
- добавлено в версии v7.0.11
