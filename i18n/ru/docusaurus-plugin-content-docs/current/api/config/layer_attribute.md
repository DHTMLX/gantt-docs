# layer_attribute
:::info
 Эта функция доступна только в PRO-версии. 
:::
### Description

@short: Определяет имя атрибута для DOM-элемента слоя задачи

@signature: layer_attribute: string

### Example

~~~jsx
gantt.config.layer_attribute = "tasklayer";
~~~

**Default value:** "data-layer"
