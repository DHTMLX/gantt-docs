# show_progress

### Description

@short: Позволяет отображать прогресс внутри полос задач

@signature: show_progress: boolean

### Example

~~~jsx
gantt.config.show_progress = true;
...
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
