# scale_height

### Description

@short: Устанавливает высоту временной шкалы и заголовка сетки

@signature: scale_height: number

### Example

~~~jsx
gantt.config.scale_height = 50;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** 35
