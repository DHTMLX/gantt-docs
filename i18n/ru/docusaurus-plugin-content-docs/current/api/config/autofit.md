# autofit

### Description

@short: Автоматически подгоняет ширину колонок grid под ширину grid

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Default value:** false
