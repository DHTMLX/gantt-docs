# autofit

### Description

@short: Включает автоматическую подстройку столбцов grid под ширину grid

@signature: autofit: boolean

### Example

~~~jsx
gantt.config.autofit = true;
gantt.config.grid_width = 500;
~~~

**Значение по умолчанию:** false
