# horizontal_scroll_key

### Description

@short: Управляет тем, происходит ли горизонтальная прокрутка при использовании клавиши Shift, Alt или Meta вместе с колесиком мыши

@signature: horizontal_scroll_key: string | boolean

### Example

~~~jsx
gantt.config.horizontal_scroll_key = "altKey";
~~~

**Default value:** "shiftKey"
