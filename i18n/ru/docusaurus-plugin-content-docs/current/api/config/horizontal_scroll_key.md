# horizontal_scroll_key

### Description

@short: Включает/выключает горизонтальную прокрутку с помощью клавиш Shift|Alt|Meta + движение колесика мыши

@signature: horizontal_scroll_key: string | boolean

### Example

~~~jsx
gantt.config.horizontal_scroll_key = "altKey";
~~~

**Значение по умолчанию:** "shiftKey"
