# link_wrapper_width

### Description

@short: Определяет ширину кликабельной области вокруг ссылки

@signature: link_wrapper_width: number

### Example

~~~jsx
gantt.config.link_wrapper_width = 30;
gantt.init("gantt_here");
~~~

**Default value:** 20
