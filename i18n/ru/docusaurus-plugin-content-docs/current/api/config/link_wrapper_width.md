# link_wrapper_width

### Description

@short: Устанавливает ширину области над ссылкой, чувствительной к кликам

@signature: link_wrapper_width: number

### Example

~~~jsx
gantt.config.link_wrapper_width = 30;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** 20
