# details_on_dblclick

### Description

@short: Включает открытие лайтбокса при двойном клике на задачу

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Default value:** true
