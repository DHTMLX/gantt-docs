# details_on_dblclick

### Description

@short: «'says'» чтобы открыть lightbox после двойного клика по задаче

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
