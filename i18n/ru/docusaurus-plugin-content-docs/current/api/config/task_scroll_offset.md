# task_scroll_offset

### Description

@short: Определяет смещение (в пикселях) для ближайшей задачи от левого края timeline

@signature: task_scroll_offset: number

### Example

~~~jsx
gantt.config.task_scroll_offset = 120;
gantt.init("gantt_here");
~~~

**Default value:** 100
