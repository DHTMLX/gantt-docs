# task_scroll_offset

### Description

@short: Устанавливает смещение (в пикселях) ближайшей задачи от левой границы во временной шкале

@signature: task_scroll_offset: number

### Example

~~~jsx
gantt.config.task_scroll_offset = 120;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** 100
