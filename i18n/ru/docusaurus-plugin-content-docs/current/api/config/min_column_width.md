# min_column_width

### Description

@short: Устанавливает минимальную ширину столбца в области временной шкалы

@signature: min_column_width: number

### Example

~~~jsx
gantt.config.min_column_width = 100;

gantt.init("gantt_here");
~~~

**Значение по умолчанию:** 70
