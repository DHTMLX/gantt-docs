# min_column_width

### Description

@short: Определяет минимальную ширину колонки в области timeline

@signature: min_column_width: number

### Example

~~~jsx
gantt.config.min_column_width = 100;
 
gantt.init("gantt_here");
~~~

**Default value:** 70
