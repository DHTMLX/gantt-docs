# drag_links

### Description

@short: Позволяет создавать зависимости между задачами через перетаскивание

@signature: drag_links: boolean

### Example

~~~jsx
gantt.config.drag_links = false;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
