# drag_move

### Description

@short: Позволяет перемещать задачи методом перетаскивания

@signature: drag_move: boolean

### Example

~~~jsx
gantt.config.drag_move = false;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
