# scroll_on_click

### Description

@short: Определяет, должна ли таймлайн прокручиваться для отображения выбранной задачи при клике

@signature: scroll_on_click: boolean

### Example

~~~jsx
gantt.config.scroll_on_click= false;
gantt.init('gantt_here');
~~~

**Default value:** true
