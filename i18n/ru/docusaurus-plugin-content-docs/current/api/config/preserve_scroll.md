# preserve_scroll

### Description

@short: Сохраняет текущие вертикальные и горизонтальные позиции скролла при перерисовке gantt chart

@signature: preserve_scroll: boolean

### Example

~~~jsx
gantt.config.preserve_scroll = false;

gantt.init("gantt_here");
~~~

**Default value:** true
