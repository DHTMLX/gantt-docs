# min_duration

### Описание

@short: Устанавливает минимальную продолжительность (в миллисекундах), которую можно задать для задачи при изменении размера.

@signature: min_duration: number

### Пример

~~~jsx
gantt.config.min_duration = 24*60*60*1000; // (1 day)
~~~

### Подробности

- Значение конфигурации задаёт временной промежуток между датами начала и конца задачи (task.start_date - task.end_date), это значение не зависит от [настроек рабочего времени](guides/working-time.md) или [вычисления продолжительности](api/method/calculateduration.md).

### Связанные руководства
- [Перемещение задач вдоль временной шкалы](guides/dnd.md)

**Значение по умолчанию:** 60*60*1000, или 3600000 мс, 1 час
