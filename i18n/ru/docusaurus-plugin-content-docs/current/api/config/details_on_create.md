# details_on_create

### Description

@short: 'говорит' открыть lightbox во время создания новых событий кликом по кнопке '+'

@signature: details_on_create: boolean

### Example

~~~jsx
gantt.config.details_on_create = true;
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
