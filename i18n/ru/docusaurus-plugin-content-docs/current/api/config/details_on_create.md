# details_on_create

### Description

@short: Позволяет lightbox автоматически открываться при создании новых событий по клику на кнопку '+'.

@signature: details_on_create: boolean

### Example

~~~jsx
gantt.config.details_on_create = true;
gantt.init("gantt_here");
~~~

**Default value:** true
