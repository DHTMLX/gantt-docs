# show_links

### Описание

@short: Включает/выключает отображение связей в диаграмме Ганта

@signature: show_links: boolean

### Пример

~~~jsx
//hides all links in the Gantt chart
gantt.config.show_links = false;
 
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** true
