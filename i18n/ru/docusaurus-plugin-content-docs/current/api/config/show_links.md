# show_links

### Description

@short: Включает или отключает отображение ссылок в Gantt chart

@signature: show_links: boolean

### Example

~~~jsx
//скрывает все ссылки в Gantt chart
gantt.config.show_links = false;
 
gantt.init("gantt_here");
~~~

**Default value:** true
