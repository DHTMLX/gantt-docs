# row_height

### Description

@short: Устанавливает высоту строк таблицы по умолчанию

@signature: row_height: number

### Example

~~~jsx
gantt.config.row_height = 40;
 
gantt.init("gantt_here");
~~~

**Значение по умолчанию:** 35
