# Интерфейс Gantt-графика

Gantt-график — один из самых популярных способов отображения деятельностей, происходивших за заданный период.

- Каждая задача на диаграмме отображается в виде полосы.
- Ось X диаграммы представляет собой шкалу времени.
- Ось Y диаграммы — это грид с названиями задач и другой информацией, связанной с задачами. Первая колонка грида — это дерево, которое представляет иерархию задач. Вы можете сворачивать, открывать ветку, щелкнув по значку плюса/минуса рядом с заголовком. Обратите внимание, диаграмма Gantt отображает только задачи, открытые в дереве.

## Управление задачами

### Добавление новых задач

- Чтобы добавить корневую задачу, щелкните по значку '+' в заголовке грида.
- Чтобы добавить дочернюю задачу, щелкните по значку '+' в строке родительской задачи.

### Редактирование задач
- Чтобы отредактировать задачу, дважды щелкните по полосе на временной шкале или по записи задачи в гриде. Затем внесите изменения и нажмите: «Сохранить» — чтобы сохранить изменения, «Отмена» — чтобы отменить изменения.
- Чтобы изменить начальную (конечную) дату задачи, перетащите задачу вдоль области временной шкалы.
- Чтобы изменить продолжительность задачи, наведите курсор на полосу задачи и перетащите ползунок прогресса.
- Чтобы изменить прогресс задачи, наведите курсор на полосу задачи и перетащите ползунок прогресса.

### Удаление задач
- Чтобы удалить задачу, дважды щелкните по полосе на временной шкале или по записи задачи в гриде. Затем нажмите «Удалить» в открытой форме редактирования.

## Управление зависимостями

### Добавление новых связей

- Чтобы добавить новую связь для задачи, наведите курсор на задачу и перетащите круглую ручку в начале (в конце) задачи к круглой ручке другой задачи.

### Удаление связей
- Чтобы удалить связь, дважды щелкните по связи и в окне подтверждения нажмите «Удалить».
