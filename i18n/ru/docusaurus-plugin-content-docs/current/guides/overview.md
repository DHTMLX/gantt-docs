Интерфейс Gantt-диаграммы
====================
Gantt-диаграмма - один из самых популярных способов отображения активности за определённый период времени.

- Каждая активность на диаграмме отображается в виде полосы.
- Ось X диаграммы - это временная шкала.
- Ось Y диаграммы - это грид с названиями активностей и другой связанной с ними информацией. Первый столбец грида - это дерево, отображающее иерархию задач. Вы можете свернуть или развернуть ветку, кликнув по значку '+'/'-' рядом с названием. Обратите внимание, что Gantt-диаграмма отображает только те задачи, которые раскрыты в дереве.

Управление задачами
----------------------
### Добавление новых задач

- Чтобы добавить корневую задачу, нажмите на иконку "+" в заголовке грида.
- Чтобы добавить подзадачу, нажмите на иконку "+" в строке родительской задачи.

### Редактирование задач
- Чтобы отредактировать задачу, дважды кликните по полосе на временной шкале или по записи задачи в гриде. Затем внесите изменения и нажмите: "Save" - чтобы сохранить изменения, "Cancel" - чтобы отменить изменения.
- Чтобы изменить дату начала (окончания) задачи, перетащите задачу по области временной шкалы.
- Чтобы изменить длительность задачи, наведите курсор на полосу задачи и перетащите её за левую или правую границу.
- Чтобы изменить прогресс выполнения задачи, наведите курсор на полосу задачи и перетащите индикатор прогресса.

### Удаление задач
- Чтобы удалить задачу, дважды кликните по полосе на временной шкале или по записи задачи в гриде. Затем нажмите "Delete" в открывшейся форме редактирования.

Управление связями между задачами
-------------------------------

### Добавление новых связей

- Чтобы добавить новую связь для задачи, наведите курсор на задачу и перетащите круглой маркер в начале (конце) задачи к круглому маркеру другой задачи.


### Удаление связей
- Чтобы удалить связь, дважды кликните по ней и нажмите "Delete" в окне подтверждения.
