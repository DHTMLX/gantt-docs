Что нового в модуле экспорта проектов MSP
===========================================

## 2.1.0.0

* Добавлена поддержка пользовательских календарей как при импорте, так и при экспорте файлов MSP и Primavera, включая календари ресурсов.

* Модуль теперь поддерживает получение значений назначения ресурсов на основе единиц при импорте файлов.

* Теперь появилась возможность задавать пользовательские значения единиц назначения ресурсов при экспорте файлов.

* Для экспорта в Primavera добавлено свойство **RemainingDuration**, позволяющее задачам иметь согласованные даты окончания без дополнительной настройки.

## 2.0.2.0

* Библиотека MPXJ обновлена до версии 11.5.4.

* Добавлена поддержка пользовательских свойств сводных задач (Summary, WBS) как для импорта, так и для экспорта в Primavera. Это работает даже если свойства имеют одинаковые имена. Чтобы включить это для сводных задач при экспорте, требуется возвращать *true* для свойства **Summary**.

* Исправлена проблема, из-за которой Baseline 0 импортировался некорректно.

## 2.0.0.1

* Проведены различные внутренние оптимизации и добавлен импортированный Dockerfile.

## 2.0.0.0

* Модуль был перенесён с ASP.NET MVC на ASP.NET Core, что позволяет запускать его на Linux и в контейнерах Docker.
