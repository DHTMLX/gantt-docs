# Сторонние лицензии


В этой статье приведены уведомления и дополнительные условия, связанные со сторонними программными компонентами, включёнными в библиотеку DHTMLX Gantt.

Скомпилированная библиотека **Bluebird**, распространяемая по лицензии MIT, включена в JS-файлы нашего Gantt. Ссылки на неё можно найти в комментариях внутри JS-файлов. Подробнее о том, как она интегрирована с Gantt, смотрите: [Promise](api/method/promise.md).

**Шрифт Roboto Google**, используемый в оформлении Material, по умолчанию не входит в состав библиотеки, но обычно добавляется вручную в соответствии с нашей документацией.

Также мы используем некоторые open source библиотеки в наших демонстрационных приложениях, такие как **Node.js Express module, body_parser и strip_tags** (указаны в package backend/package.json в примерах). Эти компоненты не являются частью самой библиотеки Gantt и включены только для демонстрационных целей.
