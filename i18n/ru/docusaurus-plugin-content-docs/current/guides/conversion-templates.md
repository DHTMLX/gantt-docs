# Шаблоны преобразования даты

- [parse_date](api/template/parse_date.md) - преобразует строку даты в объект Date
- [format_date](api/template/format_date.md) - преобразует объект Date в строку даты. Используется для отправки данных обратно на сервер

:::note
С версии 9.1.3 даты в формате ISO 8601 обрабатываются автоматически без необходимости переопределять эти шаблоны. См. [Загрузка дат в формате ISO](guides/loading.md#loading-dates-in-iso-format).
:::
