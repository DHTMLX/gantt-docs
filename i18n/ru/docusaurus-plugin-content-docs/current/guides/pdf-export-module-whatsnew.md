# Что нового в модуле экспорта PDF


## 0.6.6

* Незапланированные задачи теперь игнорируются при экспорте в Excel с использованием конфигурации 'visual'.

* Добавлены шаблоны для DHTMLX Gantt 9.0.

## 0.6.5

* Внесены корректировки в шаблон для DHTMLX Scheduler 7.0.

## 0.6.4

* Введён мультистраничный экспорт в один PDF-файл (диаграмма экспортируется частями согласно заданному формату, а затем страницы объединяются в один файл).

* Добавлены опции для отображения заголовков грида и временной шкалы на каждой странице при мультистраничном экспорте в одном файле.

## 0.6.3

* Обновлён базовый образ для среды Node.js до версии 20 в Docker-образах.

* Оптимизирован Dockerfile, что позволило уменьшить размер Docker-образа.

## 0.6.2

* Добавлен шаблон для DHTMLX Scheduler 7.0.

## 0.6.0

* Изменён endpoint для сервиса экспорта MS Project / Primavera P6.

* Исправлен экспорт в Excel для корректного выделения ячеек временной шкалы при использовании параметра "visual:true".

* Обновлены пакеты, используемые для сборки Docker-файла.

## 0.5.9

* Обновлена версия Node.js в Docker-образе.

* Удалён неиспользуемый файл из Docker-образа для устранения предупреждения по безопасности.

* Исправлена ошибка, из-за которой экспорт завершался неудачно, если в колонках грида присутствовал форматтер.

## 0.5.8

* Добавлена главная страница для модуля экспорта со ссылкой для тестирования его функционала (проверьте URL в выпадающем списке перед тестированием).
Пример: [https://export.dhtmlx.com](https://export.dhtmlx.com)

* Представлена новая функция: [импорт и экспорт Gantt для Node.js](guides/export-nodejs.md).

* Исправлены ошибки экспорта в Excel, связанные с параметром `visual:true`, теперь поддерживается экспорт с несколькими шкалами, разными единицами длительности и задачами, которые не обязательно должны начинаться или заканчиваться ровно по границам ячеек временной шкалы.

## 0.5.7

* Повышена совместимость с Node.js 14.

* Исправлены проблемы с пользовательскими языковыми локалями.

* Добавлен процесс "init" для Docker.

## 0.5.6

* Решены проблемы утечек памяти при работе на headless-серверах (без графического интерфейса) и в Docker.

## 0.5.5

* Улучшена общая производительность приложения внутри Docker-образа.

## 0.5.0

* PhantomJS заменён на Electron для экспорта PDF и PNG.
