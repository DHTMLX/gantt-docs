Операции с датами
==============

dhtmlxGantt предоставляет объект [date](api/other/date.md), который содержит различные методы форматирования дат. Эти методы могут быть полезны при работе с объектами даты.

В этой статье приведены основные и часто используемые методы. Полный список методов можно найти на [странице объекта date](api/other/date.md).


Преобразование объекта Date в строку
-------------------------------------------------------

Чтобы преобразовать объект Date в строку, используйте метод [date_to_str](api/other/date.md): 


*Этот метод возвращает функцию, которая форматирует объект Date в строку в соответствии с заданным шаблоном:*
~~~js
var formatFunc = gantt.date.date_to_str("%d/%m/%Y");
var date = formatFunc(new Date(2013, 05, 29)); // -> "29/06/2013"
~~~


Преобразование строки в объект Date
-----------------------------------------------------
Чтобы преобразовать строку обратно в объект Date, используйте метод [str_to_date](api/other/date.md): 


*Этот метод возвращает функцию, которая разбирает строку заданного формата в объект Date:*


Вы можете создать функцию для парсинга даты следующим образом:

~~~js
var formatFunc = gantt.date.str_to_date("%d/%m/%Y");
var date = formatFunc("29/06/2013"); // -> 29 июня 2013 00:00:00
~~~


Преобразование во время UTC
--------------------------------------
Чтобы преобразовать локальное время в UTC, используйте метод [convert_to_utc](api/other/date.md):

~~~js
//29 июня 2013 14:00 (местное время) -> 29 июня 2013 12:00 (utc)
var time = gantt.date.convert_to_utc(new Date(2013, 05, 29, 14, 00));
~~~

Добавление (вычитание) временного интервала к (из) даты
----------------------------------------------------------
Чтобы добавить или вычесть временной интервал из определённой даты, используйте метод [add](api/other/date.md):

~~~js
//добавляет 1 год к указанной дате: 29 июня 2013 -> 29 июня 2014
var newDate = gantt.date.add(new Date(2013, 05, 29), 1, 'year');
~~~


:::note
Полный набор методов для форматирования дат смотрите [здесь](api/other/date.md).
:::
