# Видео Руководства

Статья представляет краткий обзор видео-руководств, подготовленных для вашего удобства, чтобы сделать Gantt понятнее.

Следуя этим видео-урокам, вы сможете освоить такие вопросы, как настройка столбцов грида, настройка ячеек и заголовков грида, создание стандартной диаграммы Gantt на разных серверных платформах.
