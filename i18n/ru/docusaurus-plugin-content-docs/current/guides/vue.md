# Vue Gantt


:::note
В настоящее время не существует нативного Vue-компонента для DHTMLX Gantt.
Тем не менее, вы можете легко интегрировать DHTMLX Gantt в свои приложения на Vue, используя следующие ресурсы.
:::

Вот несколько полезных материалов для интеграции DHTMLX Gantt с Vue:

- [Пошаговое руководство по интеграции Gantt с Vue](integrations/vue/howtostart-vue.md)
- [Демо Vue Gantt на GitHub](https://github.com/DHTMLX/vue-gantt-demo)
- [Видеоурок по интеграции DHTMLX Gantt с Vue](https://www.youtube.com/watch?v=IgoXvVSerkE&t=10s)

Используя эти ресурсы, вы сможете без труда добавить DHTMLX Gantt в свои проекты на Vue и воспользоваться всеми его возможностями.
