# Angular Gantt

:::note
В настоящее время нет нативного компонента Angular для DHTMLX Gantt.
Тем не менее, вы все равно можете легко интегрировать DHTMLX Gantt в ваши приложения на Angular, используя приведенные ниже материалы.
:::

Ниже приведены рекомендуемые ресурсы для интеграции DHTMLX Gantt с Angular:

- [Пошаговый гид по интеграции Gantt с Angular](integrations/angular/howtostart-angular.md)
- [Демо Angular Gantt на GitHub](https://github.com/DHTMLX/angular-gantt-demo)
- [Видеоурок по интеграции DHTMLX Gantt с Angular](https://www.youtube.com/watch?v=LNVgNVfwzPE&t=3s)

Следуя этим руководствам, вы сможете эффективно использовать DHTMLX Gantt в проектах на Angular и полноценно использовать все его возможности.
