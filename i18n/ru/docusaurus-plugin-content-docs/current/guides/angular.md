Angular Gantt
==================

:::note
В настоящее время не существует нативного Angular-компонента для DHTMLX Gantt.
Тем не менее, вы всё равно можете легко интегрировать DHTMLX Gantt в ваши Angular-приложения, используя приведённые ниже материалы.
:::

Ниже представлены полезные ресурсы для включения DHTMLX Gantt в ваши проекты на Angular:

- [Пошаговое руководство по интеграции Gantt с Angular](integrations/angular/howtostart-angular.md)
- [Демо Gantt для Angular на GitHub](https://github.com/DHTMLX/angular-gantt-demo)
- [Видеоруководство по интеграции DHTMLX Gantt с Angular](https://www.youtube.com/watch?v=LNVgNVfwzPE&t=3s)

Эти материалы помогут вам пройти весь процесс использования DHTMLX Gantt в Angular и воспользоваться всем спектром его возможностей.
