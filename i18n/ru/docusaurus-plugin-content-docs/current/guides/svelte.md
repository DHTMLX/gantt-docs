# Svelte Gantt

:::note
В настоящее время не существует отдельного Svelte-компонента для DHTMLX Gantt.
Тем не менее, вы всё равно можете легко добавить DHTMLX Gantt в свои Svelte-приложения, используя перечисленные ниже ресурсы.
:::

Вот несколько полезных материалов для работы с DHTMLX Gantt в Svelte:

- [Пошаговое руководство по интеграции Gantt с Svelte](integrations/svelte/howtostart-svelte.md)
- [Демо Svelte Gantt на GitHub](https://github.com/DHTMLX/svelte-gantt-demo)

Используя эти ресурсы, вы сможете внедрить DHTMLX Gantt в свои проекты на Svelte и максимально использовать его возможности.
