# Спецификация формата даты

При указании формата даты вы можете использовать любой символ из следующего списка:

- **%y** - год в виде двузначного числа ( _00 до 99_ );
- **%Y** - год в виде четырехзначного числа ( _1900-9999_ );

- **%m** - месяц в виде числа с ведущим нулем ( _01 до 12_ );
- **%n** - месяц в виде числа без ведущего нуля ( _1 до 12_ ); 
- **%M** - месяц в виде аббревиатуры ( _Jan до Dec_ );
- **%F** - месяц в виде полного наименования ( _January до December_ );

- **%W** - номер недели по ISO-8601 в году. Недели начинаются с понедельника; 
- **%w** - номер недели, недели начинаются либо с понедельника, либо с воскресенья в зависимости от значения конфигурационного свойства [start_on_monday](api/config/start_on_monday.md) 

- **%d** - день в виде числа с ведущим нулем ( _01 до 31_ );
- **%j** - день в виде числа без ведущего нуля ( _1 до 31_ );
- **%D** - день в виде аббревиатуры ( _Вс до Сб_ );
- **%l** - день в виде полного наименования ( _Воскресенье до Субботы_ );

- **%h** - час в 12-часовом формате ( _00 до 11_ );
- **%H** - час в 24-часовом формате ( _00 до 23_ );
- **%g** - час в 12-часовом формате без ведущего нуля ( _1 до 12_ );
- **%G** - час в 24-часовом формате без ведущего нуля ( _0 до 23_ );

- **%i** - минута в виде числа с ведущим нулем ( _00 до 59_ );
- **%s** - секунда в виде числа с ведущим нулем ( _00 до 59_ );
- **%a** - отображает **am** (для времени с полуночи до полудня) и **pm** (для времени с полудня до полуночи);
- **%A** - отображает **AM** (для времени с полуночи до полудня) и **PM** (для времени с полудня до полуночи).

Например, если вы хотите представить 1 июня 2019 года как 01/06/2019, следует указать "%d/%m/%Y".
