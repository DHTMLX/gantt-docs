# Спецификация формата даты

При настройке форматов даты вы можете использовать следующие символы:

- **%y** - год из двух цифр ( _00 до 99_ );
- **%Y** - год из четырех цифр ( _1900-9999_ );


- **%m** - месяц в виде числа с ведущим нулем ( _01 до 12_ );
- **%n** - месяц в виде числа без ведущего нуля ( _1 до 12_ );
- **%M** - сокращенное название месяца ( _Jan до Dec_ );
- **%F** - полное название месяца ( _January до December_ );


- **%W** - номер недели года по ISO-8601, недели начинаются с понедельника;
- **%w** - номер недели, где начало недели зависит от настройки [start_on_monday](api/config/start_on_monday.md) (понедельник или воскресенье);


- **%d** - день месяца в виде числа с ведущим нулем ( _01 до 31_ );
- **%j** - день месяца в виде числа без ведущего нуля ( _1 до 31_ );
- **%D** - сокращенное название дня недели ( _Sun до Sat_ );
- **%l** - полное название дня недели ( _Sunday до Saturday_ );


- **%h** - часы по 12-часовому формату с ведущим нулем ( _00 до 11_ );
- **%H** - часы по 24-часовому формату с ведущим нулем ( _00 до 23_ );
- **%g** - часы по 12-часовому формату без ведущего нуля ( _1 до 12_ );
- **%G** - часы по 24-часовому формату без ведущего нуля ( _0 до 23_ );


- **%i** - минуты с ведущим нулем ( _00 до 59_ );
- **%s** - секунды с ведущим нулем ( _00 до 59_ );
- **%a** - отображает **am** (от полуночи до полудня) или **pm** (от полудня до полуночи);
- **%A** - отображает **AM** (от полуночи до полудня) или **PM** (от полудня до полуночи).

Например, чтобы отобразить 1 июня 2019 года как 01/06/2019, используйте формат "%d/%m/%Y".
