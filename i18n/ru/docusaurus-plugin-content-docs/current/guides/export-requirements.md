# Сервис экспорта - Требования к системе для автономной установки

Библиотека dhtmlxGantt предоставляет возможность экспорта и импорта данных из диаграммы Gantt с использованием экспорта как онлайн-сервиса.

Вы также можете экспортировать Gantt локально, установив [сервисы экспорта](https://dhtmlx.com/docs/products/dhtmlxGantt/export.shtml) на вашем компьютере. Необходимо убедиться, что ваша система соответствует системным требованиям, чтобы использовать модули экспорта:

- PNG/PDF/Excel [требования к экспорту](guides/export-requirements.md#pdfpngexcel-service)
- MS Project/Primavera P6 [требования к импорту и экспорту](guides/export-requirements.md#import-and-export-from-ms-project-and-primavera-p6)

## Сервис PDF/PNG/Excel

### Обзор

Экспорт в PDF/PNG/Excel — кросс-платформенное приложение на Node.js, основанное на JavaScript.

Оно распространяется в виде исходного кода и образа Docker.

## Импорт и экспорт из MS Project и Primavera P6

### Обзор

Экспорт в MS Project является приложением на базе .NET Core Framework, написанным на C# и работает на Windows, macOS, Linux.

Мы можем предоставить вам исходный код, который можно разместить на вашем собственном сервере или любом облачном провайдере.
Исходный проект совместим с MS Visual Studio 2022+.
