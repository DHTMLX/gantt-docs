# Сервис экспорта - системные требования для самостоятельной установки


Библиотека dhtmlxGantt позволяет экспортировать и импортировать данные из диаграммы Gantt с помощью онлайн-сервиса экспорта.

В качестве альтернативы, экспорт диаграмм Gantt можно выполнять локально, установив [экспортные сервисы](https://dhtmlx.com/docs/products/dhtmlxGantt/export.shtml) на свой компьютер. Для корректной работы экспортных модулей убедитесь, что ваша система соответствует следующим требованиям:

- Требования для экспорта в PNG/PDF/Excel [export requirements](guides/export-requirements.md#pdfpngexcelservice)
- Требования для импорта и экспорта из MS Project/Primavera P6 [import and export requirements](guides/export-requirements.md#importandexportfrommsprojectandprimaverap6) 

## Сервис PDF/PNG/Excel

### Общая информация

Экспорт в PDF/PNG/Excel реализован как кроссплатформенное приложение на Node.js, написанное на JavaScript. 


Доступен как в виде исходного кода, так и в виде Docker-образа.

## Импорт и экспорт из MS Project и Primavera P6

### Общая информация

Экспорт в MS Project реализован как приложение на .Net Core Framework, написанное на C#, и работает на Windows, MacOS и Linux.

Исходный код доступен для развертывания на вашем собственном сервере или у любого облачного провайдера.
Этот проект совместим с MS VisualStudio 2022 и более поздними версиями.
