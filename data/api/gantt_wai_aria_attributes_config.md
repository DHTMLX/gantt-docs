wai_aria_attributes
=============

@short:
	enables WAI-ARIA support to make the component recognizable for screen readers

@type: boolean
@example:
gantt.config.wai_aria_attributes = true;
...
gantt.init("gantt_here");

@default:true

@template:	api_config
@descr:
added in version 4.1

@related:
desktop/accessibility.md#waiariaattributes
