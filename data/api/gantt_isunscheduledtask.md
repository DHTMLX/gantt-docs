isUnscheduledTask
=============


@short:
	checks if the task is unscheduled 

@params:

- task			Task			the task's object

@returns:
- value	boolean	'true' if the specified task is unscheduled, 'false' otherwise 

@example:

var isUnscheduled = gantt.isUnscheduledTask(task);
