isUnscheduledTask
=============


@short:
	checks if the task is unscheduled 

@params:

- task			object			the task's object

@returns:
- value	boolean	'true' if the specified task is unscheduled, otherwise - 'false'

@example:

var isUnscheduled = gantt.isUnscheduledTask(task);
