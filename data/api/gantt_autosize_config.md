autosize
=============
@short:forces the Gantt chart to automatically change its size to show all tasks without scrolling
	

@type: boolean, string
@values:"y" ( or true),"x", "xy"
@example:
gantt.config.autosize = "xy";

gantt.init("gantt_here");
