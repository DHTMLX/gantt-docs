license
=============

@short:
	returns the license name of dhtmlxGantt

@type:string

@returns: 
- license		string		the name of the license

@example:
console.log(gantt.license);
// -> "enterprise"


@template:	api_config
@descr:
This method returns a short license name that can be used for diagnostics. 

Possible values are:

- "gpl"
- "evaluation"
- "individual"
- "commercial"
- "enterprise"
- "ultimate"
- "site"


@changelog:
added in v6.2.2
