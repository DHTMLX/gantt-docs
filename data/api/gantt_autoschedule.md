autoSchedule
=============

@short:
	recalculates the schedule of the project

@params:


@example:

gantt.autoSchedule();


@template:	api_method
@descr:
to recalculate the schedule starting from a particular task, pass the id of the task as an argument to the autoSchedule() method:

~~~js
gantt.autoSchedule(taskId);
~~~
