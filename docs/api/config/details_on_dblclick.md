# details_on_dblclick

### Description

@short: 'says' to open the lightbox after double clicking on a task

@signature: details_on_dblclick: boolean

### Example

~~~jsx
gantt.config.details_on_dblclick = true;

gantt.init("gantt_here");
~~~

**Default value:** true
