# cascade_delete

### Description

@short: Enables cascade deleting of nested tasks and links

@signature: cascade_delete: boolean

### Example

~~~jsx
gantt.config.cascade_delete = false;
gantt.init("gantt_here");
~~~

**Default value:** true

### Details

added in version 4.2
