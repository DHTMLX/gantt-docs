# link_wrapper_width

### Description

@short: Sets the width of the area (over the link) sensitive to clicks

@signature: link_wrapper_width: number

### Example

~~~jsx
gantt.config.link_wrapper_width = 30;
gantt.init("gantt_here");
~~~

**Default value:** 20
