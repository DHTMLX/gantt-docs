# wai_aria_attributes

### Description

@short: Enables WAI-ARIA support to make the component recognizable for screen readers

@signature: wai_aria_attributes: boolean

### Example

~~~jsx
gantt.config.wai_aria_attributes = true;
...
gantt.init("gantt_here");
~~~

**Default value:** true

### Details

added in version 4.1
